가구 유형 자동 판정

배우자·부양가족 유무와 배우자 소득으로 결정

3구간 산정 — 산 모양 곡선

소득이 올라가면 장려금이 올라갔다가 → 유지되다가 → 내려갑니다

예시: 단독가구, 연소득 1,500만원

1,500만은 평탄구간끝(900만) 초과 → 점감구간

= 165만 - (1,500만-900만) x 165만 / (2,200만-900만)

= 165만 - 600만 x 165만 / 1,300만 = 165만 - 76.2만

= 약 88.8만원
