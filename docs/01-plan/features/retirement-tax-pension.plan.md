## 2. 퇴직소득세 계산 로직 (nodong.kr 분석 결과)

### 2.1 계산 흐름

```
퇴직급여 → 근속연수공제 → 환산급여 → 환산급여별공제 → 과세표준 → 세율적용 → 퇴직소득세
```

### 2.2 근속연수공제 테이블

| 근속연수 | 공제금액 |
|---------|---------|
| 1~5년 | 연 100만원 |
| 6~10년 | 500만원 + (초과연수 x 200만원) |
| 11~20년 | 1,500만원 + (초과연수 x 250만원) |
| 21년~ | 4,000만원 + (초과연수 x 300만원) |

### 2.3 환산급여 계산

```
환산급여 = (퇴직급여 - 근속연수공제) x 12 / 근속연수
```

### 2.4 환산급여별 공제 테이블

| 환산급여 | 공제금액 |
|---------|---------|
| ~800만원 | 전액 공제 |
| 800만~7,000만원 | 800만 + (초과분 x 60%) |
| 7,000만~1억원 | 4,520만 + (초과분 x 55%) |
| 1억~3억원 | 6,170만 + (초과분 x 45%) |
| 3억원~ | 1억5,170만 + (초과분 x 35%) |

### 2.5 세율 구간 (종합소득세와 동일)

| 과세표준 | 세율 | 누진공제 |
|---------|------|---------|
| ~1,400만원 | 6% | 0 |
| ~5,000만원 | 15% | 126만원 |
| ~8,800만원 | 24% | 576만원 |
| ~1.5억원 | 35% | 1,544만원 |
| ~3억원 | 38% | 1,994만원 |
| ~5억원 | 40% | 2,594만원 |
| ~10억원 | 42% | 3,594만원 |
| 10억원~ | 45% | 6,594만원 |

### 2.6 최종 퇴직소득세

```
환산산출세액 = 과세표준 x 세율 - 누진공제
퇴직소득세 = floor(환산산출세액 / 12 x 근속연수)
지방소득세 = floor(퇴직소득세 x 10%)
총 세액 = 퇴직소득세 + 지방소득세
```

### 2.7 과세이연 (IRP 지급 시)

```
이연 퇴직소득세 = floor(퇴직소득세 x IRP지급액 / 퇴직급여)
원천징수 퇴직소득세 = 퇴직소득세 - 이연 퇴직소득세
```

### 3.5 severance.py 개선 사항

1. **상여금 가산**: `annual_bonus_total x 3/12`를 3개월 임금총액에 가산
2. **연차수당 가산**: `unused_annual_leave_pay x 3/12`를 3개월 임금총액에 가산
3. **IRP 의무 안내**: 2022.4 이후 퇴직자 IRP 세전 지급 의무 warning 추가
4. **퇴직소득세 연동**: severance 결과에서 retirement_tax 자동 호출 옵션

### 3.6 퇴직연금 계산기 설계

**DB형 (확정급여형):**
- 퇴직금과 동일 공식 (평균임금 x 30 x 재직연수)
- 차이: 적립금 운용은 회사 책임

**DC형 (확정기여형):**
- 매년 연간임금총액의 1/12 이상 적립
- 총 수령액 = sum(연도별 연간임금총액 / 12) + 운용수익
- 운용수익은 입력값으로 받거나 0% 기준 최소 보장
