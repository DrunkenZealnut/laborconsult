# Plan: 직장 내 괴롭힘 판정 및 해설

## Executive Summary

| 관점 | 내용 |
|------|------|
| **Problem** | 사용자가 직장 내 괴롭힘 상황을 설명하면, 현재 챗봇은 일반적인 RAG 검색만 수행하여 법적 3요소(지위 우위/업무 적정범위 초과/고통·근무환경 악화) 충족 여부를 체계적으로 판정하지 못한다. |
| **Solution** | 근로기준법 제76조의2의 3요소 판정 프레임워크를 구조화하여, 사용자 상황에서 핵심 사실관계를 추출하고 각 요소별 해당 여부를 평가하는 판정 엔진 + RAG 기반 행정해석/판례 참조 시스템을 구축한다. |
| **Function UX Effect** | "이게 괴롭힘인가요?" 질문에 대해 3요소 체크리스트 기반 구조화된 판정 결과 + 관련 행정해석 인용 + 대응 절차 안내를 제공한다. |
| **Core Value** | 직장 내 괴롭힘 피해 근로자가 자신의 상황을 법적 프레임워크로 이해하고, 적절한 대응 절차(신고, 조사 요청, 노동청 진정 등)를 안내받을 수 있다. |

---

## 1. 배경 및 문제 정의

### 1.1 현재 상태

- 챗봇은 "직장 내 괴롭힘" 관련 질문을 받으면 Pinecone 벡터 검색으로 유사한 Q&A를 찾아 Claude에게 전달
- 임금계산기처럼 구조화된 분석 없이 일반 RAG 답변만 생성
- 사용자의 구체적 상황에 대한 **법적 3요소 충족 여부 판정**이 체계적으로 이루어지지 않음

### 1.2 목표

- **근로기준법 제76조의2** 기반 3요소 판정 프레임워크를 구조화
- 사용자 상황에서 핵심 사실관계를 자동 추출
- 각 요소별 해당/비해당/불분명 판정 + 근거 제시
- 관련 행정해석(40건) 및 판례 자동 참조
- 신고/구제 절차 안내

## 2. 법적 프레임워크

### 2.1 직장 내 괴롭힘 정의 (근로기준법 제76조의2)

> 사용자 또는 근로자는 직장에서의 지위 또는 관계 등의 우위를 이용하여 업무상 적정 범위를 넘어 다른 근로자에게 신체적·정신적 고통을 주거나 근무환경을 악화시키는 행위를 하여서는 아니 된다.

### 2.2 3요소 판정 기준

| 요소 | 법적 기준 | 판정 포인트 |
|------|----------|------------|
| **① 지위·관계 우위** | 직장에서의 지위 또는 관계 등의 우위를 이용 | 직급, 정규직/비정규직, 인원수, 연령, 학벌, 성별, 출신 등 |
| **② 업무상 적정범위 초과** | 업무상 적정 범위를 넘는 행위 | 업무 지시 vs 사적 지시, 합리적 범위, 반복성, 통상적 관행 여부 |
| **③ 고통·근무환경 악화** | 신체적·정신적 고통 또는 근무환경 악화 | 정신적 스트레스, 신체적 폭력, 업무 배제, 따돌림, 사직 강요 등 |

### 2.3 괴롭힘 유형 분류 (고용노동부 매뉴얼 기반)

| 유형 | 구체적 행위 예시 |
|------|----------------|
| **폭행·협박** | 물리적 폭력, 물건 투척, 위협적 언행 |
| **폭언·모욕** | 욕설, 인격 모독, 비하 발언, 외모 비하 |
| **따돌림·무시** | 의도적 배제, 대화 무시, 정보 차단, 소문 유포 |
| **부당한 업무 지시** | 허드렛일 강요, 과도한 업무 부여, 업무 배제, 전문성 무시 |
| **사적 용무 강요** | 개인 심부름, 사적 모임 강요, 종교·정치 활동 강요 |
| **감시·통제** | 과도한 보고 요구, 사생활 감시, SNS 감시 |
| **부당인사** | 부당 전보, 승진 누락, 연봉 차별, 해고 압박 |

### 2.4 관련 법률 조문

| 조문 | 내용 |
|------|------|
| 근로기준법 제76조의2 | 직장 내 괴롭힘의 금지 (정의) |
| 근로기준법 제76조의3 | 직장 내 괴롭힘 발생 시 조치 (신고→조사→조치) |
| 근로기준법 제109조 제2항 | 불리한 처우 시 벌칙 (3년 이하 징역 또는 3천만원 이하 벌금) |
| 산업안전보건법 제41조 | 고객의 폭언 등으로 인한 건강장해 예방조치 |

### 2.5 기존 데이터 자산

- **행정해석 40건**: `output_노동부 행정해석/직장내괴롭힘/` — 가해자 범위, 조사 절차, 비밀유지, 분리조치, 피해자 보호 등
- **Pinecone BEST Q&A**: 직장 내 괴롭힘 관련 상담 사례 (벡터 검색으로 접근 가능)

## 3. 기능 설계 방향

### 3.1 핵심 설계 원칙

이 기능은 임금계산기(수치 계산)와 근본적으로 다르다. **법적 판단**은 주관적 요소가 크므로:

1. **확정 판결이 아닌 판정 가이드** — "괴롭힘에 해당합니다/아닙니다" 단정이 아닌 "해당 가능성이 높습니다/낮습니다" + 근거 제시
2. **3요소별 체크리스트** — 각 요소의 충족/미충족/불분명을 구분하여 사용자가 자신의 상황을 법적으로 이해할 수 있게 함
3. **행정해석·판례 참조** — RAG로 유사 사례를 찾아 객관적 근거를 보강
4. **면책 문구 필수** — "법적 조언이 아닌 정보 제공이며, 정확한 판단은 노동청(1350) 또는 노무사에게 문의하세요"

### 3.2 기술적 접근 방식: Structured Assessment + RAG

```
사용자 질문 → [1단계] 사실관계 추출 (Claude tool_use)
           → [2단계] 3요소 판정 엔진 (규칙 기반 + Claude 판단)
           → [3단계] Pinecone 검색 (관련 행정해석·Q&A)
           → [4단계] 종합 답변 생성 (판정 결과 + 해설 + 절차 안내)
```

**임금계산기와의 차이점:**

| 구분 | 임금계산기 | 괴롭힘 판정 |
|------|----------|-----------|
| 성격 | 수치 계산 (결정적) | 법적 평가 (확률적) |
| 엔진 | 공식 기반 계산 | 프레임워크 기반 평가 |
| 결과 | 정확한 금액 | 가능성 수준 (높음/보통/낮음) |
| RAG 역할 | 보충 설명 | 핵심 근거 |

### 3.3 모듈 구조

```
harassment_assessor/
├── __init__.py              # Public API exports
├── models.py                # HarassmentInput, 열거형 (RelationType, HarassmentType 등)
├── assessor.py              # assess_harassment() — 3요소 판정 핵심 로직
├── constants.py             # 판정 기준표, 유형별 키워드, 법 조문 참조
└── result.py                # AssessmentResult, format_assessment()
```

### 3.4 입력 파라미터 (사실관계 추출)

Claude tool_use로 사용자 질문에서 추출할 핵심 사실관계:

| 파라미터 | 설명 | 예시 |
|---------|------|------|
| `perpetrator_role` | 가해자 직위/관계 | "팀장", "동료", "고객" |
| `victim_role` | 피해자 직위/관계 | "팀원", "계약직", "신입" |
| `relationship_type` | 관계 유형 | 상급자→하급자, 동료→동료, 고객→근로자 |
| `behavior_description` | 행위 내용 | 자유 텍스트 |
| `behavior_types` | 행위 유형(복수) | ["폭언", "따돌림", "부당업무"] |
| `frequency` | 빈도 | "1회", "수회", "매일", "수개월간" |
| `duration` | 기간 | "1개월", "6개월", "1년 이상" |
| `witnesses` | 목격자 유무 | true/false |
| `evidence` | 증거 유무 | "녹음", "문자", "이메일", "없음" |
| `impact` | 피해 결과 | "우울증", "퇴사", "부서 이동", "업무 배제" |
| `company_response` | 회사 대응 | "미조치", "조사 중", "가해자 징계" |
| `business_size` | 사업장 규모 | "5인미만", "5인이상" 등 |

### 3.5 출력 구조

```
[판정 결과]
  ① 지위·관계 우위: ✅ 해당 (팀장→팀원, 직급 상 우위)
  ② 업무 적정범위 초과: ✅ 해당 (인격 모독적 폭언, 업무와 무관한 비하)
  ③ 고통·근무환경 악화: ✅ 해당 (정신적 고통, 우울증 진단)

  → 직장 내 괴롭힘 해당 가능성: 높음

[관련 법 조문]
  - 근로기준법 제76조의2 (직장 내 괴롭힘 금지)
  - 근로기준법 제76조의3 (발생 시 조치)

[유사 행정해석]
  - 근로기준정책과-XXXX: (관련 사례 요약)

[대응 절차 안내]
  1단계: 증거 확보 (녹음, 문자, 이메일 등)
  2단계: 사내 신고 (인사부서 또는 괴롭힘 고충처리위원회)
  3단계: 회사 조사 및 조치 의무 (근기법 제76조의3)
  4단계: 미조치 시 → 노동청 진정 (1350)
  5단계: 불리한 처우 시 → 형사 고소 가능 (제109조)

⚠️ 본 판정은 법적 효력이 없는 참고 정보입니다.
   정확한 판단은 노동청(1350) 또는 노무사에게 문의하세요.
```

## 5. 테스트 시나리오

| # | 시나리오 | 기대 판정 |
|---|---------|----------|
| 1 | 팀장이 팀원에게 매일 욕설 + 업무 배제 | 가능성 높음 (3요소 모두 충족) |
| 2 | 동료가 1회 험한 말 | 가능성 낮음 (우위 관계 불분명, 1회성) |
| 3 | 사장이 직원에게 사적 심부름 반복 강요 | 가능성 높음 (지위 우위 + 업무 외 행위) |
| 4 | 선배가 후배에게 업무상 엄격한 지도 | 가능성 낮음 (업무 적정범위 내) |
| 5 | 고객이 콜센터 직원에게 폭언 | 직장 내 괴롭힘 비해당 → 산안법 제41조 안내 |
| 6 | 5인 미만 사업장에서 사장 폭언 | 가능성 높음 + 5인 미만 적용 안내 |
| 7 | 동기 3명이 1명을 조직적으로 따돌림 6개월 | 가능성 높음 (인원수 우위 + 반복·지속) |
