# Plan: 근로장려금(EITC) 수급기준 및 금액 계산기

## Executive Summary

| Perspective | Description |
|-------------|-------------|
| **Problem** | 저소득 근로자가 근로장려금 수급 자격 여부와 예상 금액을 쉽게 확인할 수 없어, 신청 기한을 놓치거나 수급 가능성을 인지하지 못하는 문제 |
| **Solution** | 조세특례제한법 제100조의2~12 기반 근로장려금 계산기를 기존 wage_calculator 패키지에 추가하여, 가구유형/소득/재산 입력만으로 수급 자격 판정 + 예상 금액 산출 |
| **Function UX Effect** | 챗봇에서 "근로장려금 받을 수 있나요?" 질문 시 소득·가구 정보 입력 → 즉시 수급 여부 + 예상 금액 + 신청 안내 제공 |
| **Core Value** | 기존 19개 임금계산기에 EITC를 추가하여, 저소득 근로자 지원 범위를 임금 계산 → 세제 혜택까지 확장 |

## 1. Feature Overview

### 1.1 배경

근로장려금(EITC, Earned Income Tax Credit)은 저소득 근로자·사업자·종교인에게 근로를 장려하고 실질소득을 지원하는 환급형 세액공제 제도이다. 많은 수급 대상자가 자격 여부를 모르거나 계산 방법을 몰라 신청하지 못하는 경우가 빈번하다.

### 1.2 목표

- 가구유형·총소득·재산 입력 → 수급 자격 판정 (적격/부적격/감액)
- 점증·평탄·점감 3구간 산식에 따른 정확한 예상 금액 산출
- 자녀장려금 병행 계산 (선택)
- 챗봇(chatbot.py) 및 웹 API(pipeline.py)에서 자연어 질문으로 자동 호출

### 1.3 범위

**In Scope:**
- 근로장려금 수급 자격 판정 (소득요건 + 재산요건)
- 가구유형별(단독/홑벌이/맞벌이) 금액 산출
- 재산 감액(1.7억~2.4억 → 50%) 반영
- 자녀장려금 금액 산출 (18세 미만 부양자녀)
- 연도별 기준금액 상수 관리 (2024~2026)
- facade.py, CALC_TYPE_MAP, chatbot 연동

**Out of Scope:**
- 반기 신청 vs 정기 신청 스케줄 관리
- 사업소득·종교인 소득에 대한 별도 산식 (근로소득 기준으로 통일)
- 국세청 홈택스 API 연동

## 3. Calculation Logic

### 3.1 수급 요건

#### 소득 요건 (총소득 기준, 연간)

| 가구유형 | 2024 | 2025 | 2026 |
|---------|------|------|------|
| 단독가구 | 2,200만원 미만 | 2,200만원 미만 | 추후 확인 (잠정 동일) |
| 홑벌이가구 | 3,200만원 미만 | 3,200만원 미만 | 추후 확인 |
| 맞벌이가구 | 3,800만원 미만 | 3,800만원 미만 | 추후 확인 |

#### 재산 요건

- 가구원 전체 재산 합계 **2.4억원 미만**
- 1.7억~2.4억원: 근로장려금 **50% 감액**
- 2.4억원 이상: 수급 불가

#### 가구유형 분류

| 유형 | 기준 |
|------|------|
| 단독가구 | 배우자·부양자녀(18세 미만)·70세 이상 직계존속 없음 |
| 홑벌이가구 | 배우자 총급여 300만원 미만, 또는 부양자녀/70세+ 직계존속 있음 |
| 맞벌이가구 | 배우자 총급여 300만원 이상 |

### 3.2 금액 산출 (점증-평탄-점감 3구간)

#### 단독가구

| 구간 | 총소득 범위 | 산식 | 최대 |
|------|-----------|------|------|
| 점증 | 0 ~ 400만원 | 소득 × (165/400) | 165만원 |
| 평탄 | 400 ~ 900만원 | 165만원 | 165만원 |
| 점감 | 900 ~ 2,200만원 | 165 - (소득-900) × 165/1,300 | 0원 |

#### 홑벌이가구

| 구간 | 총소득 범위 | 산식 | 최대 |
|------|-----------|------|------|
| 점증 | 0 ~ 700만원 | 소득 × (285/700) | 285만원 |
| 평탄 | 700 ~ 1,400만원 | 285만원 | 285만원 |
| 점감 | 1,400 ~ 3,200만원 | 285 - (소득-1,400) × 285/1,800 | 0원 |

#### 맞벌이가구

| 구간 | 총소득 범위 | 산식 | 최대 |
|------|-----------|------|------|
| 점증 | 0 ~ 800만원 | 소득 × (330/800) | 330만원 |
| 평탄 | 800 ~ 1,700만원 | 330만원 | 330만원 |
| 점감 | 1,700 ~ 3,800만원 | 330 - (소득-1,700) × 330/2,100 | 0원 |

### 3.3 자녀장려금 (선택)

- 대상: 18세 미만 부양자녀
- 소득 요건: 홑벌이 4,000만원, 맞벌이 4,000만원 미만
- 자녀 1인당 최대 **100만원** (2025년 기준)
- 재산 감액 동일 적용
