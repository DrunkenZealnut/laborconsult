## 1. Problem Statement

### 현재 상태 (AS-IS)

- 챗봇 답변 말미에 "고용노동부(1350) 또는 노무사에게 문의하세요"만 반복
- 실업급여 신청, 구직활동, 직업훈련, 취업지원 등 고용센터에서 직접 처리하는 업무도 "1350"으로만 안내
- 사용자가 거주지 관할 고용센터를 직접 검색해야 하는 불편
- 기존 `labor_offices.py`는 노동위원회(14개소) 전용 — 고용센터(132개소)는 별개 기관

### 목표 상태 (TO-BE)

- 전국 132개 고용센터 연락처 데이터 내장 (본센터 + 하부 센터/출장소)
- 실업급여·구직·직업훈련 등 고용센터 소관 사안 질문 시 관할 고용센터 자동 안내
- 사용자가 지역을 언급하면 해당 지역 관할 고용센터 연락처 + 홈페이지 URL 제공
- 지역 미언급 시 1350 안내 + 고용24(work24.go.kr) 고용센터 검색 안내

### 근거

- work24.go.kr (고용24) 공식 사이트에서 전국 132개 고용센터 확인 완료
- 고용센터는 실업급여·구직활동·직업훈련 등의 현장 접수/상담 기관 — 1350 전화상담과는 별개

## 3. Scope

### In-Scope

1. **고용센터 데이터 모듈** (`employment_centers.py`): 132개 센터 연락처 + 관할구역 매핑 + 홈페이지 URL
2. **관할구역 자동 매칭**: 사용자 질문에서 지역명 추출 → 해당 고용센터 안내
3. **챗봇 시스템 프롬프트 업데이트**: 고용센터 소관 사안 판단 시 연락처 포함 지시
4. **파이프라인 통합**: 실업급여·구직·직업훈련 관련 답변에 관할 고용센터 정보 자동 삽입
5. **홈페이지 URL 제공**: 각 센터별 홈페이지 또는 고용24 검색 링크 포함

### Out-of-Scope

- 고용센터 운영시간 실시간 조회
- 온라인 예약/접수 연동
- 지도 기반 위치 안내 UI
- 고용노동부 지방관서(근로감독관) 연락처 (별도 feature)
- 고용센터별 제공 프로그램 상세 안내

## 4. Technical Approach

### 4.1 데이터 구조

```python
@dataclass
class EmploymentCenter:
    name: str               # 센터명
    phone: str              # 대표전화
    address: str            # 주소
    jurisdiction: list[str] # 관할 구·군·시
    parent: str | None      # 상위 센터명 (하부센터인 경우)
    homepage: str | None    # 개별 홈페이지 URL
```

### 4.2 기존 labor_offices.py와의 관계

| 항목 | 노동위원회 (labor_offices.py) | 고용센터 (employment_centers.py) |
|------|-----|-----|
| 기관 수 | 14개 | 132개 |
| 소관 사안 | 부당해고 구제신청, 차별시정, 노동쟁의 | 실업급여, 구직활동, 직업훈련, 취업지원 |
| 트리거 키워드 | 해고, 구제신청, 부당노동행위 | 실업급여, 구직, 직업훈련, 취업, 고용센터 |
| 데이터 모듈 | `app/core/labor_offices.py` | `app/core/employment_centers.py` (신규) |

### 4.3 관할구역 매핑 전략

- 132개 센터의 관할 시·군·구를 `REGION_MAP` dict로 매핑
- 광역시·도 이름으로 대표 센터 매핑 (예: "부산" → 부산고용센터)
- 구·군 이름으로 세부 센터 매핑 (예: "해운대" → 부산동부고용센터)
- 하부 센터는 parent 필드로 상위 센터 연결

### 4.4 트리거 조건

고용센터 연락처 제공이 필요한 사안:
- 실업급여 (구직급여) 신청·수급
- 구직활동 지원·구직등록
- 직업훈련 (내일배움카드, 국비훈련)
- 취업지원 프로그램
- 고용보험 가입·피보험자격
- 고용안정사업 (고용유지지원금 등)
- 모성보호 (출산전후휴가급여, 육아휴직급여)
- 고용센터 방문·위치 문의
