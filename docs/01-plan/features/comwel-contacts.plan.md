# Plan: 근로복지공단 연락처와 홈페이지 정보 제공

## Executive Summary

| 관점 | 내용 |
|------|------|
| **Problem** | 산재보험·체불임금보장·근로복지대부 등 근로복지공단 소관 사안에 대해 관할 지사 연락처를 안내하지 못함 |
| **Solution** | 전국 63개소(7 지역본부 + 56 지사) 연락처·주소·관할구역 데이터 모듈 + 파이프라인 연동 |
| **Function UX Effect** | "산재보험 신청하려면?" 질문 시 관할 근로복지공단 지사 전화번호·주소를 즉시 안내 |
| **Core Value** | 노동위원회(해고)·고용센터(실업급여)에 이어 근로복지공단(산재·체불보장)까지 3대 기관 안내 완성 |

---

## 1. 문제 정의

현재 챗봇은 노동위원회(14개, 해고·차별시정)와 고용센터(133개, 실업급여·직업훈련) 연락처를 제공하지만, **근로복지공단** 소관 업무에 대해서는 구체적 지사 연락처를 안내하지 못한다.

### 근로복지공단 소관 업무
| 업무 | 설명 |
|------|------|
| 산재보험 | 요양급여·휴업급여·장해급여·유족급여·장의비·간병급여 |
| 체불 임금 보장 | 체불사업주 대신 미지급 임금·퇴직금 지급 (체당금) |
| 근로복지 대부 | 생활안정자금·주거안정자금 저리 대출 (연 1.5%) |
| 퇴직연금 | 퇴직연금 적립금 운용·지급 |
| 직업재활 | 산재근로자 의료·직업·사회심리재활 |
| 근로자 건강센터 | 50인 미만 소규모 사업장 근로자 건강관리 |

## 3. 기능 범위

### 3.1 포함 (In Scope)
- 63개소 지사 데이터 모듈 (`app/core/comwel_offices.py`)
- 질문에서 지역명 추출 → 관할 지사 매칭 (`find_office()`)
- 산재보험/체불임금/복지대부 등 키워드 감지 시 컨텍스트에 지사 연락처 삽입
- SYSTEM_PROMPT에 근로복지공단 안내 규칙 추가 (규칙 13)
- `legal_hints.py` 면책 문구에 근로복지공단 참조 추가
- 지역 미명시 시 대표전화(1588-0075) + 지사 찾기 링크 안내

### 3.2 제외 (Out of Scope)
- 근로복지공단 산하 병원(11개) 연락처 — 별도 기능으로 분리
- 산재 신청서 작성 자동화
- 체당금 계산기

## 4. 기술 전략

### 4.1 구현 패턴
기존 `labor_offices.py`(14개 노동위원회)·`employment_centers.py`(133개 고용센터)와 동일한 패턴:

```
데이터 모듈 (dataclass + 리스트 + REGION_MAP + 조회함수)
  → pipeline.py import + SYSTEM_PROMPT 규칙 추가 + 컨텍스트 삽입
  → legal_hints.py 면책 문구 확장
```

### 4.2 키워드 설계
`_COMWEL_KEYWORDS`: 근로복지공단 소관 업무 키워드 (기존 `_COMMISSION_KEYWORDS`·`_CENTER_KEYWORDS`와 완전 분리)

예상 키워드:
```
산재보험, 산업재해, 산재신청, 요양급여, 휴업급여, 장해급여, 유족급여,
장의비, 간병급여, 체당금, 체불임금보장, 생활안정자금, 주거안정자금,
근로복지대부, 퇴직연금, 직업재활, 근로자건강센터, 근로복지공단
```

### 4.3 키워드 충돌 분석

| 기존 키워드 세트 | 소관 기관 | 겹침 여부 |
|----------------|----------|----------|
| `_COMMISSION_KEYWORDS` (9개) | 노동위원회 | 없음 — 해고·차별·쟁의 전용 |
| `_CENTER_KEYWORDS` (17개) | 고용센터 | 없음 — 실업급여·구직·직업훈련 전용 |
| `_COMWEL_KEYWORDS` (신규) | 근로복지공단 | 없음 — 산재·체당금·복지대부 전용 |

복합 질문 (예: "산재 후 해고당해서 실업급여도 받고 싶어요") → 3개 블록 모두 독립 평가, 해당 기관 연락처 모두 삽입.

### 4.4 지역 매칭 전략
- 고용센터(133개)와 유사하게 `_METRO_KEYWORDS` 우선순위 패턴 적용
- 구·군·시 단위 > 광역시·도 이름 매칭
- 63개소이므로 지역 미명시 시 전체 나열 대신 `format_office_guide()` (1588-0075 + 지사 찾기 링크)
