# Plan: 포괄임금제 계산기 보완 (comprehensive-wage-review)

## Executive Summary

| 관점 | 내용 |
|------|------|
| **Problem** | 현재 포괄임금제 역산이 단순 총액÷시간 방식이어서, 정액급여→기본시급 역산·수당별 적정액 비교·유효성 판단이 누락됨 |
| **Solution** | nodong.kr/inclusiveWageCal 기준으로 역산 공식 보완, 유효성 판단, 수당별 과부족 비교표, 최신 판례 11건 반영 |
| **Function UX Effect** | 포괄임금 총액만 입력해도 기본시급·각 수당 적정액·부족분을 한눈에 확인 가능 |
| **Core Value** | 포괄임금제 근로자의 최저임금·수당 적정성 검증을 정확하게 수행하여 임금 부족분 청구 근거 제공 |

## 1. 현황 분석

### 1.1 현재 구현 (comprehensive.py)
- `ComprehensiveResult`: base_wage, effective_hourly, included_allowances, is_minimum_wage_ok, shortage
- **역산 로직**:
  - breakdown 있으면: `기본급 ÷ monthly_hours` → effective_hourly
  - breakdown 없으면: `총액 ÷ monthly_hours` → effective_hourly
- **문제**: 이 방식은 수당이 포함된 총액에서 기본시급을 역산하는 것이 아니라, 단순 나눗셈만 수행

### 1.2 nodong.kr 참조 (inclusiveWageCal)
- **2가지 입력 경로**: 고정 정액급여 / 기본급+수당 구분
- **역산 공식**: `총액 = 기본시급 × (기본시간 + 연장×1.5 + 야간×0.5 + 휴일8h이내×1.5 + 휴일8h초과×2.0)` → 기본시급 역산
- **유효성 판단**: 교대제·일급·시급 적용 불가, 근로시간 산정 곤란성 판단
- **11건 판례 + 2건 행정해석** 반영
- **유급휴일/무급휴무일 구분** → 월 기준시간 산출

## 3. 구현 범위

### 3.1 핵심 (Must-Have)
1. **G1: 역산 공식 보완** — 가산율 반영 총계수시간으로 기본시급 역산
   ```
   총계수시간 = 기본시간 + 연장×1.5 + 야간×0.5 + 휴일8h이내×1.5 + 휴일8h초과×2.0
   기본시급 = 정액급여 ÷ 총계수시간
   ```
2. **G2: 수당별 적정액 산출** — 역산된 기본시급 × 시간 × 가산율로 각 수당 적정액 계산, 포함액과 비교
3. **G9: 휴일근로 8h 초과분 반영** — 역산 공식에 holiday_OT × 2.0 포함
4. **G3: 유효성 판단** — 교대제·일급·시급 적용 불가 경고, 특수업종 안내
5. **G8: 5인 미만 가산수당 제외** — 5인 미만 시 가산율 0으로 역산
6. **입력 유효성(분모 0 방지)** — 총계수시간이 0 이하인 경우 계산 중단 및 사용자 오류 메시지 반환

## 6. 역산 공식 상세

### 6.1 총계수시간 산출
```
월 기본시간 = 소정근로시간(주) × (365/7/12) + 주휴시간 × (365/7/12)
월 연장계수시간 = 연장(주) × 1.5 × (365/7/12)      # 5인 미만: × 1.0
월 야간계수시간 = 야간(주) × 0.5 × (365/7/12)      # 5인 미만: × 0.0
월 휴일계수시간 = 휴일8h이내(주) × 1.5 × (365/7/12) # 5인 미만: × 1.0
월 휴일OT계수시간 = 휴일8h초과(주) × 2.0 × (365/7/12) # 5인 미만: × 1.0
총계수시간 = 기본시간 + 연장계수 + 야간계수 + 휴일계수 + 휴일OT계수
```

### 6.2 기본시급 역산
```
포괄총액(월) = monthly_wage (또는 comprehensive_breakdown 총합)
연간 상여금 가산 = annual_bonus / 12
역산 대상 총액 = 포괄총액 + 연간상여금가산

기본시급 = 역산 대상 총액 ÷ 총계수시간
```

### 6.3 수당별 적정액 산출
```
연장수당 적정액 = 기본시급 × 연장(주) × 1.5 × (365/7/12)
야간수당 적정액 = 기본시급 × 야간(주) × 0.5 × (365/7/12)
휴일수당 적정액 = 기본시급 × 휴일8h이내(주) × 1.5 × (365/7/12)
휴일OT수당 적정액 = 기본시급 × 휴일8h초과(주) × 2.0 × (365/7/12)
기본급 적정액 = 기본시급 × 기본시간
```

### 6.4 5인 미만 사업장
```
5인 미만: 가산수당(×0.5) 미적용 →
  연장 → × 1.0 (가산 없음, 통상임금으로 지급)
  야간 → × 0.0 (야간가산 미적용)
  휴일 → × 1.0 (가산 없음)
  휴일OT → × 1.0 (가산 없음)
총계수시간(5인미만) = 기본시간 + 연장×1.0 + 휴일×1.0 + 휴일OT×1.0
```
