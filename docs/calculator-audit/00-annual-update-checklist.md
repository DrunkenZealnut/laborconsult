## 갱신 절차

1. **고용노동부 고시 확인** (매년 12월~1월)
   - 최저임금, 구직급여 상한, 산재 최저보상기준, 장례비 한도
2. **보험공단 발표 확인** (매년 1월)
   - 4대보험 요율, 건강보험료 상한/하한
3. **`constants.py` 수정**
   - 해당 딕셔너리에 새 연도 키-값 추가
   - 하위 호환 상수 갱신 여부 확인
4. **테스트 실행**
   - `python3 wage_calculator_cli.py` -- 전체 32개 테스트 케이스 통과 확인
   - 새 연도 기준 테스트 케이스 추가 (해당 시)
5. **코드 리뷰**
   - 기존 fallback 로직 (`get_insurance_rates`, `get_eitc_params` 등) 동작 확인
   - 잠정값 주석 갱신
