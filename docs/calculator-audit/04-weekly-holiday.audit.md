## 4. 계산 과정 (핵심)

### Step 1: 주휴수당 발생 요건 판단

**공식**: `주_소정근로시간 = 주_소정근로일수 × 1일_소정근로시간`

- 주 소정근로시간 >= 15시간 여부 확인 (`weekly_holiday.py:56`)
- 법적 근거: 근로기준법 제18조제3항 (주 15시간 미만 단시간근로자 적용 제외)
- 코드: `is_eligible = weekly_scheduled >= WEEKLY_HOLIDAY_MIN_HOURS`

### Step 2: 개근 여부 확인

- `weekly_attendance_days` 입력 시 소정근로일수와 비교 (`weekly_holiday.py:59-65`)
- 실출근일수 < 소정근로일수 이면 주휴수당 미발생
- 법적 근거: 근기법 제55조 (소정근로일 개근 요건)

### Step 3: 주휴 인정 시간 산정 (대법원 2022다291153)

**경우 1: 주 소정근로일 >= 5일** (`weekly_holiday.py:85-91`)

```
주휴시간 = min(1일_소정근로시간, 8시간)
```

**경우 2: 주 소정근로일 < 5일** (`weekly_holiday.py:93-99`)

```
주휴시간 = min(주_소정근로시간 ÷ 5, 8시간)
```

- 법적 근거: 대법원 2025.8.14. 선고 2022다291153
- 기존 "주소정/40 × 8" 비례방식이 아닌, 주 소정근로일 수를 기준으로 산정

### Step 4: 주휴수당 계산

**공식** (`weekly_holiday.py:101-102`):

```
주_주휴수당 = 통상시급 × 주휴시간
월_주휴수당 = 주_주휴수당 × WEEKS_PER_MONTH(4.345)
```

- 코드: `weekly_pay = hourly * holiday_hours`
- 코드: `monthly_pay = weekly_pay * WEEKS_PER_MONTH`

### Step 5: 퇴직 마지막 주 주휴수당 판단

- `end_date` 입력 시에만 판단 (`weekly_holiday.py:131-157`)
- 마지막 근무일 = 퇴직일 - 1일
- 해당 주 주휴일(일요일) 산출
- **퇴직일 <= 주휴일(일요일)** 이면 마지막 주 주휴수당 미발생
- **퇴직일 > 주휴일(일요일)** 이면 마지막 주 주휴수당 발생
- 법적 근거: 고용노동부 행정해석 변경 (2021.8.4.)

## 7. 계산 예시

### 예시 1: 주 5일, 1일 8시간 근무 (월급 250만원, 통상시급 11,962원)

**Step 1**: 주 소정근로시간 = 5일 × 8시간 = 40시간 >= 15시간 → 발생

**Step 2**: 개근 가정 (weekly_attendance_days = None)

**Step 3**: 주 소정근로일 5일 >= 5일 이므로
- 주휴시간 = min(8, 8) = 8시간

**Step 4**:
- 주 주휴수당 = 11,962원 × 8시간 = 95,696원
- 월 주휴수당 = 95,696원 × 4.345 = 415,798원

### 예시 2: 주 3일, 1일 5시간 근무 (시급 10,030원)

**Step 1**: 주 소정근로시간 = 3일 × 5시간 = 15시간 >= 15시간 → 발생

**Step 2**: 개근 가정

**Step 3**: 주 소정근로일 3일 < 5일 이므로
- 주휴시간 = min(15 ÷ 5, 8) = min(3, 8) = 3시간

**Step 4**:
- 주 주휴수당 = 10,030원 × 3시간 = 30,090원
- 월 주휴수당 = 30,090원 × 4.345 = 130,741원
