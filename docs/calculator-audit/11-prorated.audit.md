# 중도입사 일할계산 감사 시트

## 1. 개요

| 항목 | 내용 |
|------|------|
| 계산기명 | 중도입사 일할계산 |
| 코드 파일 | `wage_calculator/calculators/prorated.py` |
| 함수명 | `calc_prorated(inp, ow)` |
| 적용 법조문 | 근로기준법 제43조 (임금의 지급) |
| 5인 미만 적용 여부 | 적용 (사업장 규모 무관) |

## 2. 입력 항목

| 필드명(한글) | 필드명(영문) | 타입 | 기본값 | 필수여부 |
|-------------|-------------|------|--------|---------|
| 월급 | `inp.monthly_wage` | `Optional[float]` | `None` | 조건부 (없으면 통상임금 사용) |
| 입사일 | `inp.join_date` | `Optional[str]` | `None` | 필수 |
| 첫 달 근무일수 | `inp.first_month_worked_days` | `Optional[int]` | `None` | 선택 (없으면 역일 자동 계산) |
| 월 통상임금 | `ow.monthly_ordinary_wage` | `float` | - | `calc_ordinary_wage()` 결과 |

## 3. 적용 상수

| 상수명 | 값 | 단위 | 법적 근거 | 코드 위치 |
|--------|---|------|-----------|----------|
| (해당월 총일수) | `calendar.monthrange()` 결과 | 일 | 민법상 역일 기준 | `prorated.py:56` |

본 계산기는 외부 상수를 사용하지 않으며, `calendar.monthrange()`를 통해 해당월의 총일수를 동적으로 산출한다.

## 4. 계산 과정 (핵심)

### Step 1: 기준 월급 결정
```
월기준임금 = inp.monthly_wage 또는 ow.monthly_ordinary_wage
```
- `inp.monthly_wage`가 제공되면 우선 사용, 없으면 통상임금 월액 사용
- 코드 참조: `prorated.py:39`
- 법적 근거: 근로기준법 제43조 (임금의 지급)

### Step 2: 입사일 파싱
```
입사일 = parse_date(inp.join_date)
```
- `parse_date()` 유틸리티 함수로 `"YYYY-MM-DD"` 문자열을 `date` 객체로 변환
- 파싱 실패 시 오류 결과 반환 (경고: "입사일 미입력")
- 코드 참조: `prorated.py:42-53`

### Step 3: 해당월 총일수 산출
```
해당월총일수 = calendar.monthrange(입사일.year, 입사일.month)[1]
```
- `calendar.monthrange()`는 해당 월의 (첫 요일, 총일수) 튜플 반환
- 예: 3월 = 31일, 2월(평년) = 28일, 2월(윤년) = 29일
- 코드 참조: `prorated.py:56`

### Step 4: 근무일수 산출
```
if inp.first_month_worked_days 제공:
    근무일수 = inp.first_month_worked_days
else:
    근무일수 = 해당월총일수 - 입사일.day + 1
```
- 직접 지정된 근무일수가 우선 적용
- 미지정 시 입사일부터 월말까지의 역일(曆日) 수를 자동 계산
- 코드 참조: `prorated.py:59-62`

### Step 5: 일할계산 임금 산출
```
일할임금 = 월기준임금 x 근무일수 / 해당월총일수
```
- 역일 기준 일할계산 공식
- 코드 참조: `prorated.py:65`
- 법적 근거: 근로기준법 제43조, 행정해석 (역일 기준이 실무 일반)

### Step 6: 결과 반올림
```
최종일할임금 = round(일할임금, 0)
```
- 원 단위 반올림
- 코드 참조: `prorated.py:85`

## 6. 예외 처리 / 엣지 케이스

| 조건 | 처리 | 법적 근거 |
|------|------|-----------|
| 입사일(`join_date`) 미입력 | `prorated_wage=0` 반환, 경고 "입사일 미입력" | - |
| `monthly_wage` 미입력 | 통상임금(`ow.monthly_ordinary_wage`) 대체 사용 | 근기법 제43조 |
| `first_month_worked_days` 직접 지정 | 자동 계산 대신 지정값 사용 (취업규칙 반영) | 취업규칙/근로계약 |

## 7. 계산 예시

### 예시 1: 3월 15일 입사, 월급 2,500,000원

**입력:**
- 월급: 2,500,000원
- 입사일: 2025-03-15

**계산:**
1. 해당월 총일수: 3월 = 31일
2. 근무일수: 31 - 15 + 1 = 17일
3. 일할임금: 2,500,000 x 17 / 31 = 1,370,968원

**결과:**
- `prorated_wage` = 1,370,968원
- `method` = "역일 기준"

### 예시 2: 2월 20일 입사 (윤년), 월급 3,000,000원

**입력:**
- 월급: 3,000,000원
- 입사일: 2024-02-20

**계산:**
1. 해당월 총일수: 2024년 2월 = 29일 (윤년)
2. 근무일수: 29 - 20 + 1 = 10일
3. 일할임금: 3,000,000 x 10 / 29 = 1,034,483원

**결과:**
- `prorated_wage` = 1,034,483원
- `method` = "역일 기준"
