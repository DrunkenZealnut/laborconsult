## 4. 계산 과정 (핵심)

### Step 1: 휴가 일수 결정
- 공식: `휴가일수 = 120 if 다태아 else 90`
- 코드: `maternity_leave.py:81`
- 법적 근거: 근로기준법 제74조제1항

### Step 2: 상한액 조회
- 공식: `상한액 = MATERNITY_LEAVE_UPPER[기준연도]`
- 미등록 연도: 2025년 기준 fallback
- 코드: `maternity_leave.py:89`

### Step 3: 월 급여 산정
- 공식: `월급여 = min(월통상임금, 상한액)`
- 상한 초과 시 상한액 적용
- 코드: `maternity_leave.py:93-94`

### Step 4: 최저임금 하한 보장
- 공식: `최저임금월액 = 최저시급 × 209`
- 조건: `월급여 < 최저임금월액` → `월급여 = 최저임금월액`
- 코드: `maternity_leave.py:106-112`
- 법적 근거: 근로기준법 최저임금 보장

### Step 5: 기업 유형별 부담 구조 결정
- **우선지원대상기업** (중소기업):
  - `고용보험지원일수 = 휴가일수` (전액)
  - `사업주부담일수 = 0`
  - 코드: `maternity_leave.py:115-122`
  - 법적 근거: 고용보험법 제75조제1항
- **대규모 기업**:
  - `사업주부담일수 = min(60, 휴가일수)`
  - `고용보험지원일수 = max(0, 휴가일수 - 60)`
  - 코드: `maternity_leave.py:125-131`
  - 법적 근거: 고용보험법 제75조제2항

### Step 6: 일 환산 급여 및 총액 계산
- 공식: `일급여 = 월급여 × 12 ÷ 365`
- 공식: `고용보험지급총액 = 일급여 × 고용보험지원일수`
- 공식: `사업주부담총액 = 일급여 × 사업주부담일수`
- 코드: `maternity_leave.py:138-142`

### Step 7: 배우자 출산휴가 급여 계산
- 공식: `일통상임금 = 시간당통상임금 × 1일소정근로시간`
- 공식: `배우자휴가급여 = 일통상임금 × 10일`
- 우선지원대상기업: 5일분 고용보험 지원 가능 안내
- 코드: `maternity_leave.py:156-170`
- 법적 근거: 남녀고용평등법 제18조의2

## 7. 계산 예시

### 예시 1: 우선지원대상기업, 단태아 (월 통상임금 250만원, 2025년)

**입력:** 월통상임금 = 2,500,000원, 우선지원대상기업, 단태아

| 단계 | 계산 | 결과 |
|------|------|------|
| 휴가 일수 | 단태아 | 90일 |
| 상한 비교 | min(2,500,000, 2,096,270) | 2,096,270원/월 (상한 적용) |
| 최저임금 체크 | 2,096,270 ≥ 10,030×209 = 2,096,270 | 통과 |
| 부담 구조 | 우선지원 → 전액 고용보험 | 90일 고용보험 |
| 일 급여 | 2,096,270 × 12 ÷ 365 | 68,918원/일 |
| 고용보험 총액 | 68,918 × 90 | 6,202,620원 |
| 배우자 휴가 | (2,500,000÷209) × 8 × 10 | 956,938원 |

### 예시 2: 대규모 기업, 다태아 (월 통상임금 180만원, 2025년)

**입력:** 월통상임금 = 1,800,000원, 대규모 기업, 다태아

| 단계 | 계산 | 결과 |
|------|------|------|
| 휴가 일수 | 다태아 | 120일 |
| 상한 비교 | 1,800,000 < 2,096,270 | 1,800,000원/월 (상한 미적용) |
| 부담 구조 | 대규모 → 60일 사업주, 60일 고용보험 | |
| 일 급여 | 1,800,000 × 12 ÷ 365 | 59,178원/일 |
| 사업주 부담 | 59,178 × 60 | 3,550,685원 |
| 고용보험 총액 | 59,178 × 60 | 3,550,685원 |
