# 실업급여 감사 시트

## 1. 개요

| 항목 | 내용 |
|------|------|
| 계산기명 | 실업급여(구직급여) |
| 코드 파일 | `wage_calculator/calculators/unemployment.py` |
| 함수명 | `calc_unemployment(inp, ow)` |
| 적용 법조문 | 고용보험법 제45조(수급자격), 제46조(구직급여 일액), 제50조(소정급여일수), 제64조(조기재취업수당) |
| 5인 미만 적용 여부 | 적용 (고용보험 가입 사업장이면 규모 무관) |

## 2. 입력 항목

| 필드명(한글) | 필드명(영문) | 타입 | 기본값 | 필수여부 |
|-------------|-------------|------|--------|---------|
| 월급 | `inp.monthly_wage` | `Optional[float]` | `None` | 조건부 |
| 최근 3개월 임금 | `inp.last_3m_wages` | `Optional[list]` | `None` | 조건부 (정밀 계산용) |
| 최근 3개월 총일수 | `inp.last_3m_days` | `Optional[int]` | `None` | 조건부 (last_3m_wages와 쌍) |
| 연간 상여금 총액 | `inp.annual_bonus_total` | `float` | `0.0` | 선택 |

| 통상시급 | `ow.hourly_ordinary_wage` | `float` | - | `calc_ordinary_wage()` 결과 |

## 4. 계산 과정 (핵심)

### Step 1: 피보험기간 산정
```
if inp.insurance_months 제공:
    피보험기간(개월) = inp.insurance_months
elif inp.start_date 제공:
    피보험기간(개월) = max(0, (퇴직일 - 입사일).days / 30.44)
else:
    피보험기간(개월) = 0

피보험단위기간(일) = 피보험기간(개월) x 30
```
- `30.44`는 월 평균 일수 근사값
- 코드 참조: `unemployment.py:108-117`
- 법적 근거: 고용보험법 제45조

### Step 2: 수급 자격 판단

#### 2-1. 피보험단위기간 검증
```
if 피보험단위기간(일) < 180:
    -> 수급 자격 없음 반환
```
- 이직일 이전 18개월간 180일 이상 필요
- 코드 참조: `unemployment.py:122-128`
- 법적 근거: 고용보험법 제45조

#### 2-2. 이직 사유 검증
```
if not 비자발적이직:
    if 예외사유 있음:
        -> 경고 후 계산 진행
    else:
        -> 수급 자격 없음 반환
```
- 자발적 이직 시 예외 사유 없으면 수급 불가
- 코드 참조: `unemployment.py:131-146`
- 법적 근거: 고용보험법 제58조

### Step 3: 평균임금 일액 산정
```
상여금3개월비례 = (연간상여금총액 / 12) x 3
연차수당3개월비례 = (미사용연차수당 / 12) x 3
추가3개월 = 상여금3개월비례 + 연차수당3개월비례

방법 A (last_3m_wages 제공 시):
    평균임금일액 = (sum(last_3m_wages) + 추가3개월) / last_3m_days

방법 B (monthly_wage만 제공):
    평균임금일액 = (monthly_wage x 3 + 추가3개월) / 92

방법 C (위 둘 다 없음):
    평균임금일액 = 통상시급 x 8
```
- 코드 참조: `unemployment.py:149-182`
- 법적 근거: 고용보험법 제46조, 근기법 제2조(평균임금 정의)

### Step 4: 구직급여 일액 산정
```
기준일액 = 평균임금일액 x 0.60

상한액 = UNEMPLOYMENT_BENEFIT_UPPER[기준연도]
하한액 = 최저임금(기준연도) x 0.80 x 8

if 기준일액 >= 상한액:
    실지급일액 = 상한액
elif 기준일액 <= 하한액:
    실지급일액 = 하한액
else:
    실지급일액 = 기준일액
```
- 하한액 > 상한액인 경우(2026년 이후 가능) 하한액으로 통일
- 코드 참조: `unemployment.py:185-223`
- 법적 근거: 고용보험법 제46조, 시행령 제68조

### Step 5: 소정급여일수 산정
```
연령구분 = (만나이 >= 50) 또는 장애인여부
소정급여일수 = _get_benefit_days(피보험기간(개월), 연령구분)
```
- `BENEFIT_DAYS_TABLE`에서 피보험기간에 해당하는 행을 찾아 반환
- 코드 참조: `unemployment.py:226-233`, `unemployment.py:313-318`
- 법적 근거: 고용보험법 제50조

### Step 6: 총 구직급여 산출
```
총구직급여 = 실지급일액 x 소정급여일수
```
- 코드 참조: `unemployment.py:236-239`

### Step 7: 조기재취업수당 산출
```
절반일수 = 소정급여일수 // 2
조기재취업수당 = 실지급일액 x 절반일수 x 0.50
```
- 소정급여일수의 절반 이상 남기고 취업 후 12개월 고용 유지 시 지급
- 코드 참조: `unemployment.py:243-248`
- 법적 근거: 고용보험법 제64조

## 7. 계산 예시

### 예시 1: 월급 300만원, 36세, 피보험기간 5년, 비자발적 이직 (2025년 기준)

**입력:**
- 월급: 3,000,000원
- 만 나이: 36세
- 피보험기간: 60개월
- 비자발적 이직: True
- 기준 연도: 2025

**계산:**
1. 피보험단위기간: 60 x 30 = 1,800일 >= 180일 -> 자격 충족
2. 평균임금 일액: (3,000,000 x 3) / 92 = 97,826원
3. 기준 일액: 97,826 x 60% = 58,696원
4. 상한액: 66,000원, 하한액: 10,030 x 80% x 8 = 64,192원
5. 기준 일액(58,696) < 하한액(64,192) -> 하한액 적용: 64,192원
6. 소정급여일수: 60개월, 50세 미만 -> 210일
7. 총 구직급여: 64,192 x 210 = 13,480,320원
8. 조기재취업수당: 64,192 x 105 x 50% = 3,370,080원

**결과:**
- `daily_benefit` = 64,192원 (하한액 적용)
- `benefit_days` = 210일
- `total_benefit` = 13,480,320원
- `early_reemployment_bonus` = 3,370,080원

### 예시 2: 고액 연봉자, 52세, 피보험기간 12년 (2025년 기준)

**입력:**
- 월급: 8,000,000원
- 만 나이: 52세
- 피보험기간: 144개월
- 비자발적 이직: True
- 기준 연도: 2025

**계산:**
1. 피보험단위기간: 144 x 30 = 4,320일 >= 180일 -> 자격 충족
2. 평균임금 일액: (8,000,000 x 3) / 92 = 260,870원
3. 기준 일액: 260,870 x 60% = 156,522원
4. 상한액: 66,000원 -> 상한액 적용
5. 소정급여일수: 144개월, 50세 이상 -> 270일
6. 총 구직급여: 66,000 x 270 = 17,820,000원
7. 조기재취업수당: 66,000 x 135 x 50% = 4,455,000원

**결과:**
- `daily_benefit` = 66,000원 (상한액 적용)
- `benefit_days` = 270일
- `total_benefit` = 17,820,000원
