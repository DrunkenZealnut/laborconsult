# 연차수당 계산기 감사 시트

## 1. 개요

| 항목 | 내용 |
|------|------|
| 계산기명 | 연차수당 계산기 |
| 코드 파일 | `wage_calculator/calculators/annual_leave.py` |
| 함수명 | `calc_annual_leave(inp: WageInput, ow: OrdinaryWageResult) -> AnnualLeaveResult` |
| 적용 법조문 | 근로기준법 제60조 (연차 유급휴가), 제61조 (사용촉진), 제18조 제3항 (단시간근로자), 제11조 (적용 범위) |

- `_calc_accrued_days()` (L247~288) -- 입사일 기준 연차 발생일수 산정 + G1 차감

- `_apply_part_time_ratio()` (L426~448) -- 단시간근로자 비례 연차 적용
- `_build_accrual_schedule()` (L326~383) -- 연도별 연차 발생 스케줄 생성
- `_count_complete_months()` (L453~463) -- 역월(曆月) 기준 완성 달 수 계산

## 4. 계산 과정 (핵심)

### Step 1: 재직기간 산정 (L78~90)

```
입사일 = parse_date(inp.start_date)
퇴직일 = parse_date(inp.end_date) or date.today()
재직일수 = (퇴직일 - 입사일).days
재직연수 = 재직일수 / 365
```

- **법적 근거**: 근기법 제60조
- **코드 참조**: `annual_leave.py` L78~90

### Step 2: 단시간근로자 판정 (L98~116)

```
주소정근로시간 = 1일소정근로시간 × 주소정근로일수
```

- 주 소정근로시간 < 15시간 → 연차 미발생 (0일 반환)
- 주 소정근로시간 >= 40시간 → 단시간 아님 (비례 미적용)
- **법적 근거**: 근기법 제18조 제3항
- **코드 참조**: `_apply_part_time_ratio()` L426~448

### Step 3: 연차 발생일수 산정

#### 3-A: 입사일 기준 (`_calc_accrued_days()`, L247~288)

**1년 미만** (L261~269):
```
근무월수 = _count_complete_months(입사일, 퇴직일)
발생일수 = min(근무월수, 11)
출근율 < 1.0 → 발생일수 = int(발생일수 × 출근율)
```

**1년 이상** (L271~286):
```
추가연수 = int(재직연수 - 1)
가산일수 = min(추가연수 // 2, 10)
기본일수 = min(15 + 가산일수, 25)

# G1: 제60조③ — 2년차(추가연수==0)일 때 1년 미만 사용분 차감
if 추가연수 == 0 and 1년미만사용연차 > 0:
    차감일수 = min(1년미만사용연차, 11)
    기본일수 = max(0, 기본일수 - 차감일수)

# 출근율 80% 미만 → 연차 미발생
if 출근율 < 0.8:
    기본일수 = 0
```

- **법적 근거**: 근기법 제60조 제1항, 제2항, 제3항, 제4항
- **코드 참조**: `_calc_accrued_days()` L247~288

#### 3-B: 회계기준일 기준 (`_calc_fiscal_year_leave()`, L293~321)

```
입사 첫해:
    잔여월수 = 12 - 입사월 + (1 if 입사일==1일 else 0)
    발생일수 = ceil(15 × 잔여월수 / 12)

2년째 이후:
    근속연수 = 계산연도 - 입사연도
    if 근속연수 <= 1:
        발생일수 = 15
    else:
        가산일수 = min((근속연수 - 1) // 2, 10)
        발생일수 = min(15 + 가산일수, 25)
```

- **법적 근거**: 행정해석 (회계기준일 방식 허용, 입사일 기준 이상 보장)
- **코드 참조**: `_calc_fiscal_year_leave()` L293~321

### Step 4: 단시간근로자 비례 적용 (L133~142)

```
비례계수 = (주소정근로시간 / 40) × (8 / 1일소정근로시간)
발생일수 = round(발생일수 × 비례계수, 1)
```

- **법적 근거**: 근기법 제18조 제3항
- **코드 참조**: `_apply_part_time_ratio()` L445~446

### Step 5: 미사용 연차수당 산정 (L144~171)

```
미사용일수 = max(0, 발생일수 - 사용일수)
1일통상임금 = 통상시급 × 8
```

**사용촉진제도 미실시:**
```
연차수당 = 1일통상임금 × 미사용일수
```

**사용촉진제도 실시 (`leave_use_promotion=True`):**
```
연차수당 = 0  (수당 지급 의무 면제)
```

- **법적 근거**: 근기법 제61조 (사용촉진)
- **코드 참조**: `annual_leave.py` L144~171

### Step 6: 퇴직 시 회계기준일/입사일 비교 정산 (L174~190)

```
if 회계기준일 사용:
    입사일기준발생 = _calc_accrued_days(...)
    if 입사일기준발생 > 회계기준일발생:
        차이일수 = 입사일기준발생 - 회계기준일발생
        추가지급 = 1일통상임금 × 차이일수
```

- **법적 근거**: 행정해석 (퇴직 시 입사일 기준 이상 보장)
- **코드 참조**: `annual_leave.py` L174~190

## 7. 계산 예시

### 예시 1: 일반 정규직 3년차 근로자

**입력:**
- 입사일: 2023-01-01
- 계산기준일: 2026-01-01
- 통상시급: 12,000원
- 출근율: 100%
- 사용 연차: 5일
- 1년 미만 사용 연차: 3일

**계산 과정:**

```
재직일수 = (2026-01-01 - 2023-01-01).days = 1096일
재직연수 = 1096 / 365 = 3.00년

추가연수 = int(3.00 - 1) = 2
가산일수 = min(2 // 2, 10) = 1
기본일수 = min(15 + 1, 25) = 16일

(2년차 차감은 추가연수==0일 때만 적용 → 3년차이므로 미적용)

미사용일수 = max(0, 16 - 5) = 11일
1일통상임금 = 12,000 × 8 = 96,000원
연차수당 = 96,000 × 11 = 1,056,000원
```

**결과:** 발생 16일, 미사용 11일, 연차수당 1,056,000원

### 예시 2: 단시간근로자 (주 20시간, 1일 4시간, 주 5일)

**입력:**
- 입사일: 2024-07-01
- 계산기준일: 2026-07-01
- 통상시급: 10,030원
- 1일 소정근로시간: 4시간
- 주 소정근로일수: 5일

**계산 과정:**

```
재직일수 = 731일, 재직연수 = 2.00년
주소정근로시간 = 4 × 5 = 20시간 (15시간 이상 → 연차 발생)

추가연수 = int(2.00 - 1) = 1
가산일수 = min(1 // 2, 10) = 0
기본일수 = min(15 + 0, 25) = 15일

비례계수 = (20 / 40) × (8 / 4) = 0.5 × 2.0 = 1.0
비례 발생일수 = round(15 × 1.0, 1) = 15.0일

1일통상임금 = 10,030 × 8 = 80,240원
연차수당 = 80,240 × 15.0 = 1,203,600원
```

**결과:** 발생 15.0일, 연차수당 1,203,600원
