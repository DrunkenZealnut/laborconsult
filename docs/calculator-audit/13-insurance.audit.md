# 4대보험·소득세 계산기 감사 시트

## 1. 개요

| 항목 | 내용 |
|------|------|
| 계산기명 | 4대보험료 및 근로소득세 계산기 |
| 코드 파일 | `wage_calculator/calculators/insurance.py` |
| 함수명 | `calc_insurance(inp: WageInput, ow: OrdinaryWageResult) -> InsuranceResult` |
| 보조 함수 | `calc_employer_insurance(inp, ow) -> EmployerInsuranceResult` (사업주 부담금) |
| 적용 법조문 | 국민연금법 제88조, 국민건강보험법 제69조·제76조, 고용보험법 제49조, 소득세법 제55조·제134조, 소득세법 제59조의2 (자녀세액공제), 산업재해보상보험법 제13조 |
| 5인 미만 적용 여부 | 4대보험·소득세는 사업장 규모와 무관하게 적용 (코드에서 별도 분기 없음) |

**주요 내부 함수:**
- `_calc_employee()` (L100~234) -- 근로자 4대보험 + 근로소득세 계산
- `_calc_freelancer()` (L237~305) -- 3.3% 사업소득 원천징수 계산
- `calc_employer_insurance()` (L335~468) -- 사업주 4대보험 부담금
- `_calc_child_tax_credit()` (L471~478) -- 자녀세액공제
- `_stepped_deduction()` (L504~521) -- 근로소득공제 단계별 계산
- `_calc_income_tax()` (L524~530) -- 근로소득세 누진세율 적용

## 4. 계산 과정 (핵심)

### 경로 분기 (L83~97)

```
세전월급여 = 월통상임금 (ow.monthly_ordinary_wage)

if 프리랜서:
    → _calc_freelancer()  (3.3% 원천징수)
else:
    → _calc_employee()    (4대보험 + 근로소득세)
```

---

### 경로 A: 근로자 (`_calc_employee`, L100~234)

#### Step 1: 국민연금 (L125~135)

```
기준소득월액 = max(하한, min(세전급여, 상한))
기준소득월액 = (기준소득월액 // 1000) × 1000   # 1,000원 미만 절사
국민연금 = int(기준소득월액 × 4.5%)             # 원 미만 절사
```

- 2025년: 하한 390,000원, 상한 6,170,000원
- **법적 근거**: 국민연금법 제88조
- **코드 참조**: `insurance.py` L126~128

#### Step 2: 건강보험 (L137~146)

```
건강보험 = int(세전급여 × 3.545%)               # 원 미만 절사
건강보험 = max(하한, min(건강보험, 상한))
```

- 2025년: 하한 9,890원, 상한 4,240,710원
- **법적 근거**: 국민건강보험법 제69조
- **코드 참조**: `insurance.py` L138~139

#### Step 3: 장기요양보험 (L148~152)

```
장기요양 = int(건강보험료 × 12.95%)             # 원 미만 절사
```

- **법적 근거**: 장기요양보험법 제8조
- **코드 참조**: `insurance.py` L149

#### Step 4: 고용보험 (L154~158)

```
고용보험 = int(세전급여 × 0.9%)                 # 원 미만 절사
```

- **법적 근거**: 고용보험법 제49조
- **코드 참조**: `insurance.py` L155

#### Step 5: 4대보험 합계 (L160)

```
4대보험합계 = 국민연금 + 건강보험 + 장기요양 + 고용보험
```

#### Step 6: 근로소득세 (L162~195)

```
# 6-1. 과세소득 산정
월과세소득 = max(0, 세전급여 - 월비과세소득)
연간총급여 = 월과세소득 × 12

# 6-2. 근로소득공제 (단계형)
근로소득공제 = _stepped_deduction(연간총급여)
  구간:  ~500만원: 70%
         ~1,000만원: 40%
         ~3,000만원: 15%
         ~5,500만원: 5%
         5,500만원~: 2%
  한도: 2,000만원

# 6-3. 인적공제
인적공제 = 부양가족수 × 1,500,000원

# 6-4. 과세표준
과세표준 = max(0, 연간총급여 - 근로소득공제 - 인적공제)

# 6-5. 산출세액
연간소득세 = (과세표준 × 세율 - 누진공제) - 표준세액공제(13만원)
연간소득세 = max(0, 연간소득세)

# 6-6. 자녀세액공제 적용
if 8~20세자녀 >= 1:
    자녀공제 = _calc_child_tax_credit(자녀수)
    월소득세 = max(0, 월소득세 - 자녀공제)

# 6-7. 월 세액
월소득세 = int(연간소득세 / 12)                  # 원 미만 절사
```

- **법적 근거**: 소득세법 제55조 (세율), 제47조 (근로소득공제), 제50조 (인적공제), 제59조의2 (자녀세액공제), 제134조 (원천징수)
- **코드 참조**: `insurance.py` L162~188

#### Step 7: 지방소득세 (L190)

```
지방소득세 = int(월소득세 × 10%)                # 원 미만 절사
```

- **법적 근거**: 지방세법 제89조
- **코드 참조**: `insurance.py` L190

#### Step 8: 실수령액 (L198~200)

```
세금합계 = 월소득세 + 지방소득세
총공제액 = 4대보험합계 + 세금합계
실수령액 = 세전급여 - 총공제액
```

---

### 경로 B: 프리랜서 (`_calc_freelancer`, L237~305)

```
소득세 = round(세전급여 × 3.0%)
지방소득세 = round(세전급여 × 0.3%)
원천징수합계 = 소득세 + 지방소득세    # 3.3%
실수령액 = round(세전급여 - 원천징수합계)

# 4대보험: 전부 0원
# 실질 근로자 판단 기준 7가지 안내 경고 출력
```

- **법적 근거**: 소득세법 제127조, 제145조
- **코드 참조**: `insurance.py` L249~254

---

### 경로 C: 사업주 부담금 (`calc_employer_insurance`, L335~468)

```
# 국민연금/건강보험/장기요양/고용보험: 근로자와 동일 요율
# 추가 항목:
직업능력개발 = int(세전급여 × 규모별요율)
산재보험 = int(세전급여 × (업종요율 + 출퇴근0.6% + 임금채권0.9% + 석면0.03%))
사업주합계 = 국민연금 + 건강 + 장기요양 + 고용 + 직업능력개발 + 산재
총인건비 = 세전급여 + 사업주합계
```

- **법적 근거**: 산재보험법 제13조, 고용보험법 시행령 제12조
- **코드 참조**: `insurance.py` L335~468

## 7. 계산 예시

### 예시 1: 월급 300만원 정규직 (본인 포함 부양가족 3인, 자녀 1명)

**입력:**
- 월 통상임금: 3,000,000원
- 기준 연도: 2025
- 부양가족 수: 3 (본인 + 배우자 + 자녀)
- 8~20세 자녀 수: 1
- 월 비과세: 200,000원

**계산 과정:**

```
[4대보험]
기준소득월액 = max(390,000, min(3,000,000, 6,170,000)) = 3,000,000
기준소득월액 = (3,000,000 // 1000) × 1000 = 3,000,000
국민연금 = int(3,000,000 × 4.5%) = 135,000원

건강보험 = int(3,000,000 × 3.545%) = 106,350원
건강보험 = max(9,890, min(106,350, 4,240,710)) = 106,350원

장기요양 = int(106,350 × 12.95%) = 13,772원
고용보험 = int(3,000,000 × 0.9%) = 27,000원
4대보험합계 = 135,000 + 106,350 + 13,772 + 27,000 = 282,122원

[소득세]
월과세소득 = 3,000,000 - 200,000 = 2,800,000원
연간총급여 = 2,800,000 × 12 = 33,600,000원

근로소득공제 = _stepped_deduction(33,600,000)
  = 500만 × 70% + 500만 × 40% + 2,000만 × 15% + 360만 × 5%
  = 3,500,000 + 2,000,000 + 3,000,000 + 180,000
  = 8,680,000원

인적공제 = 3 × 1,500,000 = 4,500,000원
과세표준 = max(0, 33,600,000 - 8,680,000 - 4,500,000) = 20,420,000원

연간소득세 = 20,420,000 × 15% - 1,260,000 = 1,803,000원
연간소득세 = 1,803,000 - 130,000(표준세액공제) = 1,673,000원

자녀세액공제 = 12,500원/월 (1명)
월소득세 = int(1,673,000 / 12) = 139,416원
월소득세 = max(0, 139,416 - 12,500) = 126,916원

지방소득세 = int(126,916 × 10%) = 12,691원

[합계]
세금합계 = 126,916 + 12,691 = 139,607원
총공제액 = 282,122 + 139,607 = 421,729원
실수령액 = 3,000,000 - 421,729 = 2,578,271원
```

**결과:** 실수령액 2,578,271원 (총 공제 421,729원)

### 예시 2: 프리랜서 월 400만원

**입력:**
- 세전 지급액: 4,000,000원
- `is_freelancer = True`

**계산 과정:**

```
소득세 = round(4,000,000 × 3.0%) = 120,000원
지방소득세 = round(4,000,000 × 0.3%) = 12,000원
원천징수합계 = 120,000 + 12,000 = 132,000원
실수령액 = round(4,000,000 - 132,000) = 3,868,000원

4대보험: 전부 0원
```

**결과:** 실수령액 3,868,000원 (원천징수 132,000원, 3.3%)
