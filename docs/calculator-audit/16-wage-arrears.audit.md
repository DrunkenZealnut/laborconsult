# 임금체불 지연이자 감사 시트

## 1. 개요

| 항목 | 내용 |
|------|------|
| 계산기명 | 임금체불 지연이자 |
| 코드 파일 | `wage_calculator/calculators/wage_arrears.py` |
| 함수명 | `calc_wage_arrears(arrear_amount, arrear_due_date, is_post_retirement_arrear, arrear_calc_date)` |
| 적용 법조문 | 근로기준법 제37조 (금품 청산), 상법 제54조 (상사법정이율), 근로기준법 제49조 (임금채권 소멸시효) |
| 5인 미만 적용 여부 | 적용 (사업장 규모 무관) |
| 특이사항 | 독립 함수 (WageInput 불필요, 직접 호출) |

## 2. 입력 항목

| 필드명(한글) | 필드명(영문) | 타입 | 기본값 | 필수여부 |
|-------------|-------------|------|--------|---------|
| 미지급 임금 원금 | `arrear_amount` | `float` | - | 필수 |
| 지급예정일 | `arrear_due_date` | `str` | - | 필수 ("YYYY-MM-DD") |
| 퇴직 후 14일 초과 여부 | `is_post_retirement_arrear` | `bool` | `False` | 선택 |
| 이자 계산 기준일 | `arrear_calc_date` | `Optional[str]` | `None` (오늘) | 선택 |

## 3. 적용 상수

| 상수명 | 값 | 단위 | 법적 근거 | 코드 위치 |
|--------|---|------|-----------|----------|

| 재직 중 이자율 | 0.06 (6%) | 연이율 | 상법 제54조 | `wage_arrears.py:104` |

## 4. 계산 과정 (핵심)

### Step 1: 날짜 파싱
```
지급예정일 = datetime.strptime(arrear_due_date, "%Y-%m-%d").date()
계산기준일 = datetime.strptime(arrear_calc_date, "%Y-%m-%d").date() 또는 오늘
```
- 파싱 실패 시 오늘 날짜로 대체, 경고 추가
- 코드 참조: `wage_arrears.py:64-77`

### Step 2: 소멸시효 산정
```
소멸시효만료일 = 지급예정일.replace(year=지급예정일.year + 3)
소멸시효만료여부 = 계산기준일 > 소멸시효만료일
```
- 윤년(2월 29일) 처리: `replace()` 실패 시 `timedelta(days=365*3)` 대체
- 만료 시 경고: "법원에 시효중단 조치 없이는 임금채권 행사 불가할 수 있습니다"
- 코드 참조: `wage_arrears.py:80-94`
- 법적 근거: 근로기준법 제49조 (임금채권 소멸시효 3년)

### Step 3: 이자율 결정
```
if 퇴직후14일초과:
    이자율 = 0.20 (연 20%)
    법적근거 = "근로기준법 제37조"
else:
    이자율 = 0.06 (연 6%)
    법적근거 = "상법 제54조"
```
- `is_post_retirement_arrear=True`이면 이미 퇴직 후 14일 초과로 가정
- 코드 참조: `wage_arrears.py:97-106`
- 법적 근거: 근기법 제37조 (퇴직 후 14일 초과), 상법 제54조 (재직 중)

### Step 4: 지연일수 계산
```
지연일수 = max(0, (계산기준일 - 지급예정일).days)
```
- 음수 방지: 계산기준일이 지급예정일 이전이면 0일
- 코드 참조: `wage_arrears.py:111`

### Step 5: 지연이자 계산
```
지연이자 = 미지급원금 x 이자율 x (지연일수 / 365)
```
- 일할계산: 연이율을 365일로 나누어 일별 이자 산출
- 코드 참조: `wage_arrears.py:114`
- 법적 근거: 근기법 제37조 (지연이자)

### Step 6: 총 청구액 산출
```
총청구액 = 미지급원금 + 지연이자
```
- 코드 참조: `wage_arrears.py:115`

### Step 7: 결과 반올림
```
지연이자 = round(지연이자, 0)
총청구액 = round(총청구액, 0)
```
- 코드 참조: `wage_arrears.py:157-158`

## 7. 계산 예시

### 예시 1: 퇴직 후 임금체불 (연 20%), 180일 지연

**입력:**
- 미지급 원금: 5,000,000원
- 지급예정일: 2025-06-15
- 퇴직 후 14일 초과: True
- 계산기준일: 2025-12-12 (180일 후)

**계산:**
1. 이자율: 20% (퇴직 후 14일 초과)
2. 지연일수: (2025-12-12) - (2025-06-15) = 180일
3. 지연이자: 5,000,000 x 0.20 x (180 / 365) = 493,151원
4. 총 청구액: 5,000,000 + 493,151 = 5,493,151원
5. 소멸시효: 2028-06-15 (미만료)

**결과:**
- `interest_amount` = 493,151원
- `total_claim` = 5,493,151원
- `is_expired` = False

### 예시 2: 재직 중 임금체불 (연 6%), 365일 지연

**입력:**
- 미지급 원금: 3,000,000원
- 지급예정일: 2024-03-10
- 퇴직 후 14일 초과: False
- 계산기준일: 2025-03-10 (365일 후)

**계산:**
1. 이자율: 6% (재직 중)
2. 지연일수: 365일
3. 지연이자: 3,000,000 x 0.06 x (365 / 365) = 180,000원
4. 총 청구액: 3,000,000 + 180,000 = 3,180,000원
5. 소멸시효: 2027-03-10 (미만료)

**결과:**
- `interest_amount` = 180,000원
- `total_claim` = 3,180,000원
- `is_expired` = False
