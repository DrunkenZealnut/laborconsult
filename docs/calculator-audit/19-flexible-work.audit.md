# 탄력적 근로시간제 감사 시트

## 1. 개요

| 항목 | 내용 |
|------|------|
| 계산기명 | 탄력적 근로시간제 연장수당 계산기 |
| 코드 파일 | `wage_calculator/calculators/flexible_work.py` |
| 함수명 | `calc_flexible_work(inp: WageInput, ow: OrdinaryWageResult) -> FlexibleWorkResult` |
| 적용 법조문 | 근로기준법 제51조(2주 단위), 제51조의2(3개월 단위), 제51조의3(6개월 단위), 제51조의4(연속휴식 보장) |
| 5인 미만 적용 여부 | 제도 자체는 적용 가능하나 가산수당은 미적용 (근기법 제56조 적용 제외) |

## 2. 입력 항목

| 필드명(한글) | 필드명(영문) | 타입 | 기본값 | 필수여부 |
|-------------|-------------|------|--------|---------|
| 탄력근무 단위기간 | `flexible_work_unit` | `str` | `""` | 필수 |
| 주별 실근로시간 리스트 | `weekly_hours_list` | `list[float]` | `None` | 선택 |
| 사업장 규모 | `business_size` | `BusinessSize` | `OVER_5` | 선택 |
| 1일 소정근로시간 | `schedule.daily_work_hours` | `float` | `8.0` | 선택 (리스트 없을 때 추정용) |
| 주 소정근로일수 | `schedule.weekly_work_days` | `float` | `5.0` | 선택 (리스트 없을 때 추정용) |
| 주 연장근로시간 | `schedule.weekly_overtime_hours` | `float` | `0.0` | 선택 (리스트 없을 때 추정용) |
| 시간당 통상임금 | `ow.hourly_ordinary_wage` | `float` | - | 필수 (OrdinaryWageResult에서 수령) |

| 상수명 | 값 | 단위 | 법적 근거 | 코드 위치 |

| `FLEXIBLE_WORK_LIMITS["3개월"]` | weeks=13, max_per_week=52, max_avg=40 | 시간 | 근기법 제51조의2 | `flexible_work.py:30` |
| `FLEXIBLE_WORK_LIMITS["6개월"]` | weeks=26, max_per_week=52, max_avg=40 | 시간 | 근기법 제51조의3 | `flexible_work.py:31` |

## 4. 계산 과정 (핵심)

### Step 1: 단위기간 설정
- 유효값: `"2주"`, `"3개월"`, `"6개월"`
- 미설정/유효하지 않은 값 → `"2주"` 기본 적용 + 경고
- 코드: `flexible_work.py:78-80`

### Step 2: 주별 실근로시간 데이터 처리
- **리스트 미입력 시**: 스케줄 기반 추정
  - `주실근로 = 1일소정근로 × 주근로일 + 주연장근로`
  - `리스트 = [주실근로] × 단위주수`
  - 코드: `flexible_work.py:88-95`
- **리스트 길이 < 단위주수**: 평균값으로 패딩 + 경고
  - 코드: `flexible_work.py:98-103`
- **리스트 길이 > 단위주수**: 초과분 절삭
  - 코드: `flexible_work.py:104-105`

### Step 3: 탄력제 기준 연장근로 판단
- 공식: `법정기준시간 = 주평균기준(40) × 단위주수`
- 공식: `총실근로 = sum(주별실근로리스트)`
- 공식: `연장근로시간 = max(0, 총실근로 - 법정기준시간)`
- 코드: `flexible_work.py:107-124`
- 법적 근거: 근기법 제51조 (단위기간 내 주당 평균 40시간 초과분)

### Step 4: 주별 최대 근로한도 초과 체크
- 2주 단위: 주당 최대 48시간
- 3/6개월 단위: 주당 최대 52시간
- 공식: `초과주수 = count(주실근로 > 주당최대한도)`
- 공식: `초과시간합계 = sum(max(0, 주실근로 - 주당최대한도))`
- 코드: `flexible_work.py:127-128`

### Step 5: 추가 가산수당 산출
- 5인 이상 사업장: `추가가산수당 = 시간급 × 초과시간합계 × OVERTIME_RATE(0.5)`
- 5인 미만 사업장: `추가가산수당 = 0` (가산 미적용)
- 코드: `flexible_work.py:133`

### Step 6: 단위기간 연장수당 산출
- **5인 이상**: `연장수당 = 시간급 × 연장근로시간 × (1 + OVERTIME_RATE)` = × 1.5
- **5인 미만**: `연장수당 = 시간급 × 연장근로시간 × 1.0` (가산 없음)
- 코드: `flexible_work.py:146-150`

### Step 7: 월 환산
- 공식: `월환산개월 = 단위주수 ÷ WEEKS_PER_MONTH`
- 공식: `월수당 = (연장수당 + 추가가산수당) ÷ 월환산개월`
- 코드: `flexible_work.py:160-162`
- 주의: 2주 단위 시 `월환산개월 ≈ 0.46`이므로 나누면 월 기준 증폭됨
- 주의: `months_per_period`에 `max(1, ...)` 가드 없음 — 단, `WEEKS_PER_MONTH > 0` 보장으로 0 나눗셈 불가

## 5. 출력 항목

| 항목명 | 설명 | 단위 |
|--------|------|------|
| `unit_period` | 단위기간 ("2주"/"3개월"/"6개월") | 문자열 |
| `unit_weeks` | 단위기간 주 수 | 주 |
| `total_actual_hours` | 단위기간 총 실근로시간 | 시간 |
| `legal_hours` | 법정 기준 총시간 (40h × 주수) | 시간 |
| `overtime_hours` | 연장근로시간 (총 초과분) | 시간 |
| `overtime_pay_per_period` | 단위기간 연장수당 합계 | 원 |
| `monthly_overtime_pay` | 월 환산 연장수당 | 원 |
| `weeks_exceeding_limit` | 주별 최대 한도 초과 주 수 | 주 |
| `extra_premium_pay` | 추가 가산수당 | 원 |

## 7. 계산 예시

### 예시 1: 2주 단위 탄력제 (시간급 15,000원, 5인 이상)

**입력:** 단위기간 = "2주", 주별실근로 = [50, 34], 시간급 = 15,000원

| 단계 | 계산 | 결과 |
|------|------|------|
| 법정기준 | 40 × 2주 | 80시간 |
| 총실근로 | 50 + 34 | 84시간 |
| 연장근로 | 84 - 80 | 4시간 |
| 주별한도 초과 | 제1주 50h > 48h → 2h 초과 | 1주 초과 |
| 연장수당 | 15,000 × 4h × 1.5 | 90,000원 |
| 추가가산 | 15,000 × 2h × 0.5 | 15,000원 |
| 월환산개월 | 2 ÷ 4.345 | 0.46개월 |
| 월수당 | (90,000 + 15,000) ÷ 0.46 | 228,261원/월 |

### 예시 2: 3개월 단위, 5인 미만 사업장

**입력:** 단위기간 = "3개월", 주별 평균 42h × 13주, 시간급 = 12,000원, 5인 미만

| 단계 | 계산 | 결과 |
|------|------|------|
| 법정기준 | 40 × 13주 | 520시간 |
| 총실근로 | 42 × 13 | 546시간 |
| 연장근로 | 546 - 520 | 26시간 |
| 연장수당 | 12,000 × 26h × 1.0 (가산 없음) | 312,000원 |
| 월환산개월 | 13 ÷ 4.345 | 2.99개월 |
| 월수당 | 312,000 ÷ 2.99 | 104,348원/월 |
