# 연장/야간/휴일수당 계산기 감사 시트

## 1. 개요

| 항목 | 내용 |
|------|------|
| 계산기명 | 연장/야간/휴일수당 (Overtime Pay) |
| 코드 파일 | `wage_calculator/calculators/overtime.py` |
| 함수명 | `calc_overtime(inp: WageInput, ow: OrdinaryWageResult) -> OvertimeResult` |
| 보조 함수 | `check_weekly_hours_compliance(inp: WageInput) -> WeeklyHoursComplianceResult` |
| 적용 법조문 | 근로기준법 제56조 (연장/야간/휴일 근로), 제53조 (연장 근로의 제한), 제52조 (선택적 근로시간제) |
| 5인 미만 적용 여부 | 부분 적용 — 5인 미만은 가산수당 미적용(1.0배만), 주 52시간 한도 미적용 |

수요: 954건 (36.7%) — 가장 많은 임금 관련 질문 유형.

## 4. 계산 과정 (핵심)

### Step 1: 가산율 결정 (5인 이상/미만 분기)

> overtime.py:54-66

```
5인 이상 사업장:
  연장가산율 = 0.5 (OVERTIME_RATE)
  야간가산율 = 0.5 (NIGHT_PREMIUM_RATE)
  휴일가산율 = 0.5 (HOLIDAY_RATE)
  휴일초과가산율 = 0.5 (HOLIDAY_OT_RATE)

5인 미만 사업장:
  연장가산율 = 0.0
  야간가산율 = 0.0
  휴일가산율 = 0.0
  휴일초과가산율 = 0.0
```
- 법적 근거: 근기법 제11조 (5인 미만 사업장 적용 제외)

### Step 2: 연장수당 산출

> overtime.py:68-76

```
연장수당(주) = 통상시급 x 주연장근로시간 x (1 + 연장가산율)
```
- 5인 이상: `통상시급 x 연장시간 x 1.5`
- 5인 미만: `통상시급 x 연장시간 x 1.0` (기본임금만, 가산 없음)

### Step 3: 야간수당 산출

> overtime.py:77-83

```
야간수당(주) = 통상시급 x 주야간근로시간 x 야간가산율
```
- 야간(22~06시)은 연장/휴일과 **중복 가산** (통상임금의 0.5배 추가)
- 5인 미만: 야간가산 미적용 (야간수당 = 0)

### Step 4: 휴일수당 산출

> overtime.py:85-102

```
휴일수당(주) = 통상시급 x 휴일근로시간(8h이내) x (1 + 휴일가산율)
             + 통상시급 x 휴일근로초과시간 x (1 + 휴일가산율 + 휴일초과가산율)
```

| 구분 | 5인 이상 배율 | 5인 미만 배율 |
|------|-------------|-------------|
| 휴일 8h 이내 | 1.5배 | 1.0배 |
| 휴일 8h 초과 | 2.0배 (1 + 0.5 + 0.5) | 1.0배 |

### Step 5: 주 → 월 환산

> overtime.py:104-105

```
월수당합계 = (연장수당(주) + 야간수당(주) + 휴일수당(주)) x WEEKS_PER_MONTH
```
- `WEEKS_PER_MONTH` = 365 / 7 / 12 ≈ 4.345

### Step 6: 주 52시간 초과 경고

> overtime.py:107-111

```
주총근로시간 = (1일소정근로시간 x 주소정근로일수) + 주연장시간 + 주휴일시간 + 주휴일초과시간
```
- 52시간 초과 시 경고 메시지 출력

### check_weekly_hours_compliance() — 주 52시간 준수 체크

> overtime.py:147-232

별도 함수로 상세한 준수 여부를 체크한다:

```
소정근로 = 1일소정근로시간 x 주소정근로일수
연장근로 = 주연장근로시간
휴일근로 = 주휴일근로시간 + 주휴일근로초과시간
주총근로 = 소정근로 + 연장근로 + 휴일근로

초과시간 = max(0, 주총근로 - 52)
```

| 조건 | 결과 | 제재 |
|------|------|------|
| 5인 미만 | 미적용 (is_applicable=False) | 없음 |
| 52h 이하 | 준수 (is_compliant=True) | 없음 |
| 52h ~ 60h | 위반 + 특별연장 안내 | 2년 이하 징역 또는 2,000만원 이하 벌금 (제110조) |
| 60h 초과 | 위반 | 2년 이하 징역 또는 2,000만원 이하 벌금 (제110조) |

## 7. 계산 예시

### 예시 1: 5인 이상 사업장, 연장+야간 근무

**입력**:
- 통상시급: 15,000원 (통상임금 계산기에서 산출)
- 사업장 규모: 5인 이상
- 주 연장근로: 10시간
- 주 야간근로: 4시간 (22~06시)
- 주 휴일근로: 0시간

**계산 과정**:

Step 1 — 가산율:
```
연장가산율 = 0.5 (5인 이상)
야간가산율 = 0.5
```

Step 2 — 연장수당:
```
연장수당(주) = 15,000 x 10h x (1 + 0.5) = 15,000 x 10 x 1.5 = 225,000원/주
```

Step 3 — 야간수당:
```
야간수당(주) = 15,000 x 4h x 0.5 = 30,000원/주 (중복 가산분)
```

Step 4 — 휴일수당:
```
휴일수당(주) = 0원
```

Step 5 — 월 환산:
```
월수당 = (225,000 + 30,000 + 0) x 4.345 = 1,107,975원/월
```

### 예시 2: 5인 미만 사업장, 휴일근로

**입력**:
- 통상시급: 10,030원
- 사업장 규모: 5인 미만
- 주 연장근로: 5시간
- 주 휴일근로: 10시간 (8h + 2h 초과)

**계산 과정**:

Step 1 — 가산율:
```
5인 미만 → 모든 가산율 = 0.0
```

Step 2~4 — 수당 산출:
```
연장수당(주) = 10,030 x 5h x (1 + 0.0) = 50,150원/주
야간수당(주) = 0원
휴일수당(주) = 10,030 x 8h x (1 + 0.0) + 10,030 x 2h x (1 + 0.0 + 0.0)
             = 80,240 + 20,060 = 100,300원/주
```

Step 5 — 월 환산:
```
월수당 = (50,150 + 0 + 100,300) x 4.345 = 653,455원/월
```

참고: 5인 미만이므로 가산수당 없이 기본임금만 지급. 주 52시간 한도 미적용.
