# 근로장려금(EITC) 및 자녀장려금 감사 시트

## 1. 개요

| 항목 | 내용 |
|------|------|
| 계산기명 | 근로장려금(EITC) 및 자녀장려금 계산기 |
| 코드 파일 | `wage_calculator/calculators/eitc.py` |
| 함수명 | `calc_eitc(inp: WageInput, ow: OrdinaryWageResult) -> EitcResult` |
| 적용 법조문 | 조세특례제한법 제100조의3(수급 요건), 제100조의5(근로장려금 산정), 제100조의27(자녀장려금) |
| 5인 미만 적용 여부 | 해당 없음 (세법 기반 제도, 사업장 규모 무관) |
| 특이사항 | `ow`(OrdinaryWageResult)는 직접 사용하지 않음 — 인터페이스 통일 목적으로만 수령 |

## 4. 계산 과정 (핵심)

### Step 1: 가구유형 결정
- `household_type` 유효값: `"단독"`, `"홑벌이"`, `"맞벌이"`
- 미입력/유효하지 않은 값 → `_determine_household_type(inp)` 자동 판정 + 경고
- **자동 판정 기준** (`eitc.py:255-265`):

| 조건 | 판정 결과 |
|------|----------|
| 배우자 없음 + 자녀 없음 + 직계존속 없음 | 단독 |
| 배우자 있음 + 배우자 소득 >= 300만원 | 맞벌이 |
| 그 외 (배우자 or 자녀 or 존속 있음) | 홑벌이 |

- 배우자 유무 판정: `spouse_income > 0` 또는 `tax_dependents > 1` (`eitc.py:259`)
- 코드: `eitc.py:94-104`

### Step 2: 연간 총소득 결정
- 우선순위:
  1. `annual_total_income > 0` → 그대로 사용
  2. `monthly_wage` 존재 → `monthly_wage × 12` 추정 + 경고
  3. `annual_wage` 존재 → 그대로 사용
  4. 모두 없음 → 수급 불가 반환 (`_ineligible`)
- 음수 보정: `income = max(0.0, annual_total_income)` (`eitc.py:108`)
- 코드: `eitc.py:107-124`

### Step 3: EITC 기준표 조회
- `get_eitc_params(year)` → 해당 연도 가구유형별 기준표
- 등록 연도: 2024, 2025 (그 외 연도는 가장 최근 기준 사용 + 경고)
- 코드: `eitc.py:126-130`

### Step 4: 소득 요건 체크
- 공식: `income >= phase_out_end(소득 상한)` → 수급 불가
- 코드: `eitc.py:132-138`

### Step 5: 재산 요건 체크
- 공식: `assets >= 2.4억(EITC_ASSET_UPPER)` → 수급 불가
- 음수 보정: `assets = max(0.0, total_assets)` (`eitc.py:141`)
- 코드: `eitc.py:140-146`

### Step 6: 근로장려금 3구간 산출 (`_calc_eitc_amount`)
- **점증 구간** (0 ~ `inc_end`):
  - 공식: `eitc = income × (max_amt / inc_end)`
  - 코드: `eitc.py:275-276`
- **평탄 구간** (`inc_end` ~ `flat_end`):
  - 공식: `eitc = max_amt` (최대금액 고정)
  - 코드: `eitc.py:278-279`
- **점감 구간** (`flat_end` ~ `phase_out_end`):
  - 공식: `eitc = max_amt - (income - flat_end) × max_amt / (phase_out_end - flat_end)`
  - 음수 방지: `max(0.0, amount)`
  - 코드: `eitc.py:281-284`
- 최종: `round(eitc_raw)` 원 단위 반올림 (`eitc.py:150`)
- 법적 근거: 조세특례제한법 제100조의5

### Step 7: 재산 감액
- 조건: `assets > 1.7억(EITC_ASSET_REDUCTION)` and `assets < 2.4억`
- 공식: `eitc_final = round(eitc_raw × (1 - 0.50))` = 50% 감액
- 미해당 시: `eitc_final = eitc_raw`
- 재산 미입력 (`assets == 0`): 재산 요건 미검증 경고 (계산은 감액 없이 수행)
- 코드: `eitc.py:157-177`

### Step 8: 자녀장려금 산출
- **전제 조건**: `num_children > 0` and `household != "단독"` and `income < 4,000만원`
- 연도별 `CHILD_CREDIT_MAX_PER_CHILD` 조회 (미등록 시 최근 연도 fallback) (`eitc.py:186-188`)
- 재산 감액 적용 시: `child_per = round(child_max × (1 - 0.50))` = 500,000원
- 미적용 시: `child_per = child_max` = 1,000,000원
- 공식: `child_total = child_per × num_children`
- 코드: `eitc.py:179-203`
- 법적 근거: 조세특례제한법 제100조의27

### Step 9: 합계
- 공식: `total_credit = eitc_final + child_total`
- 코드: `eitc.py:206-207`

## 7. 계산 예시

### 예시 1: 홑벌이가구, 점감 구간, 재산 감액 + 자녀장려금

**입력:** 가구유형 = 홑벌이, 연간 총소득 = 15,000,000원, 재산 = 180,000,000원(1.8억), 자녀 2명

| 단계 | 계산 | 결과 |
|------|------|------|
| 가구유형 확인 | 홑벌이 → 최대 2,850,000원 | |
| 소득구간 판정 | 15,000,000 > 14,000,000(평탄종료) | 점감 구간 |
| 근로장려금 (점감) | 2,850,000 - (15,000,000 - 14,000,000) × 2,850,000 / (32,000,000 - 14,000,000) | 2,691,667원 |
| 재산 감액 | 1.8억 > 1.7억 → 50% 감액 | |
| 근로장려금 (최종) | round(2,691,667 × 50%) | 1,345,834원 |
| 자녀장려금 (1명당) | 1,000,000 × 50% (재산 감액) | 500,000원 |
| 자녀장려금 합계 | 500,000 × 2명 | 1,000,000원 |
| 총 합계 | 1,345,834 + 1,000,000 | 2,345,834원 |

### 예시 2: 단독가구, 점증 구간, 재산 감액 없음

**입력:** 가구유형 = 단독, 연간 총소득 = 3,000,000원, 재산 = 100,000,000원(1억)

| 단계 | 계산 | 결과 |
|------|------|------|
| 가구유형 확인 | 단독 → 최대 1,650,000원 | |
| 소득구간 판정 | 3,000,000 ≤ 4,000,000(점증종료) | 점증 구간 |
| 근로장려금 (점증) | round(3,000,000 × 1,650,000 / 4,000,000) | 1,237,500원 |
| 재산 감액 | 1억 ≤ 1.7억 → 감액 미적용 | |
| 근로장려금 (최종) | | 1,237,500원 |
| 자녀장려금 | 단독가구 → 대상 아님 | 0원 |
| 총 합계 | 1,237,500 + 0 | 1,237,500원 |

### 예시 3: 맞벌이가구, 평탄 구간

**입력:** 가구유형 = 맞벌이, 연간 총소득 = 12,000,000원, 재산 = 50,000,000원(5천만), 자녀 1명

| 단계 | 계산 | 결과 |
|------|------|------|
| 가구유형 확인 | 맞벌이 → 최대 3,300,000원 | |
| 소득구간 판정 | 8,000,000 ≤ 12,000,000 ≤ 17,000,000 | 평탄 구간 |
| 근로장려금 | 최대금액 고정 | 3,300,000원 |
| 재산 감액 | 5천만 ≤ 1.7억 → 미적용 | |
| 자녀장려금 | 12,000,000 < 40,000,000 → 대상 | 1,000,000원 |
| 총 합계 | 3,300,000 + 1,000,000 | 4,300,000원 |
