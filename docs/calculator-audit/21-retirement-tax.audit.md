## 4. 계산 과정 (핵심)

### Step 1: 퇴직급여 및 근속일수 결정
- **SeveranceResult 있을 때**: 퇴직금과 재직일수를 SeveranceResult에서 가져옴
- **SeveranceResult 없을 때**: `retirement_pay_amount` 사용, `start_date`/`end_date`로 재직일수 산출
- 퇴직급여 ≤ 0 또는 재직일수 ≤ 0이면 `_zero_result` 반환 (세액 0)
- 코드: `retirement_tax.py:76-86`

### Step 2: 근속연수 계산
- 공식: `총월수 = 재직일수 × 12 ÷ 365`
- 제외/가산 반영: `총월수 = 총월수 - 제외월수 + 가산월수`
- 공식: `근속연수 = max(1, ceil(총월수 / 12))` (1년 미만 올림)
- 코드: `retirement_tax.py:89-91`

### Step 3: 근속연수공제
- `_calc_service_deduction(근속연수)` — 구간별 누진 공제
- 상한: `근속연수공제 = min(공제액, 퇴직급여)` (퇴직급여를 초과할 수 없음)
- 코드: `retirement_tax.py:96-97`, `retirement_tax.py:222-229`
- 법적 근거: 소득세법 제48조

### Step 4: 환산급여
- 공식: `환산급여 = floor((퇴직급여 - 근속연수공제) × 12 ÷ 근속연수)`
- 음수 방지: `환산급여 = max(0, 환산급여)`
- 코드: `retirement_tax.py:102-105`
- 법적 근거: 소득세법 시행령 제42조의2

### Step 5: 환산급여별 공제
- `_calc_converted_deduction(환산급여)` — 5단계 누진 공제
- 코드: `retirement_tax.py:113`, `retirement_tax.py:232-241`

### Step 6: 과세표준
- 공식: `과세표준 = max(0, 환산급여 - 환산급여별공제)`
- 코드: `retirement_tax.py:118`

### Step 7: 환산산출세액
- `_calc_tax_by_brackets(과세표준)` — 종합소득세율 적용
- 공식: `세액 = 과세표준 × 세율 - 누진공제`
- 코드: `retirement_tax.py:126`, `retirement_tax.py:244-251`

### Step 8: 최종 퇴직소득세 및 지방소득세
- 공식: `퇴직소득세 = floor(환산산출세액 ÷ 12 × 근속연수)`
- 공식: `지방소득세 = floor(퇴직소득세 × 10%)`
- 공식: `총세액 = 퇴직소득세 + 지방소득세`
- 코드: `retirement_tax.py:131-133`

### Step 9: IRP 과세이연 (선택)
- 조건: `irp_transfer_amount > 0` and `retirement_pay > 0`
- 공식: `이연퇴직소득세 = floor(퇴직소득세 × IRP이체금액 ÷ 퇴직급여)`
- 공식: `이연지방소득세 = floor(이연퇴직소득세 × 10%)`
- 원천징수: `원천징수소득세 = floor((퇴직소득세 - 이연세액) / 10) × 10` (10원 미만 절사)
- 코드: `retirement_tax.py:142-163`
- 법적 근거: 근로자퇴직급여보장법 제9조

### Step 10: 실수령 퇴직금
- 공식: `실수령액 = 퇴직급여 - 원천징수소득세 - 원천징수지방소득세`
- 코드: `retirement_tax.py:166`

## 6. 예외 처리 / 엣지 케이스

| 조건 | 처리 | 법적 근거 |
|------|------|-----------|
| 퇴직급여 ≤ 0 | `_zero_result` 반환 + "퇴직급여가 0원 이하" 경고 | `retirement_tax.py:85-86,254-271` |
| 근속일수 ≤ 0 | `_zero_result` 반환 + "근속기간 확인 불가" 경고 | `retirement_tax.py:85-86,254-271` |
| `end_date` 미입력 | `date.today()` 사용 | `retirement_tax.py:82` |
| 근속연수 < 1년 | `max(1, ceil(...))` → 최소 1년 보장 | `retirement_tax.py:91` |
| 근속연수공제 > 퇴직급여 | `min(공제액, 퇴직급여)` → 퇴직급여 한도 | `retirement_tax.py:97` |
| 환산급여 < 0 | `max(0, ...)` → 0원 처리 | `retirement_tax.py:105` |
| 과세표준 < 0 | `max(0, ...)` → 0원 처리 | `retirement_tax.py:118` |
| IRP 이체금액 > 퇴직급여 | `min(IRP, 퇴직급여)` → 퇴직급여 한도 | `retirement_tax.py:142` |
| `retirement_pay == 0` (IRP 비율 계산) | 조건문으로 0 나눗셈 방지 | `retirement_tax.py:143` |
| 원천징수액 10원 미만 | `floor(... / 10) × 10` 절사 처리 | `retirement_tax.py:158-163` |
| SeveranceResult 우선 사용 | SeveranceResult 제공 시 `retirement_pay_amount` 무시 | `retirement_tax.py:76-78` |
| `start_date` 미입력 (severance 없을 때) | `parse_date` 실패 → `service_days = 0` → 0 결과 | `retirement_tax.py:82-83` |

## 7. 계산 예시

### 예시 1: 퇴직금 5,000만원, 근속 10년 (IRP 없음)

**입력:** 퇴직급여 = 50,000,000원, 근속 10년

| 단계 | 계산 | 결과 |
|------|------|------|
| 근속연수공제 | 5,000,000 + (10-5)×2,000,000 | 15,000,000원 |
| 환산급여 | floor((50,000,000 - 15,000,000) × 12 / 10) | 42,000,000원 |
| 환산급여별공제 | 8,000,000 + (42,000,000 - 8,000,000) × 60% | 28,400,000원 |
| 과세표준 | 42,000,000 - 28,400,000 | 13,600,000원 |
| 환산산출세액 | 13,600,000 × 6% | 816,000원 |
| 퇴직소득세 | floor(816,000 / 12 × 10) | 680,000원 |
| 지방소득세 | floor(680,000 × 10%) | 68,000원 |
| 총 세액 | 680,000 + 68,000 | 748,000원 |
| 실수령액 | 50,000,000 - 680,000 - 68,000 | 49,252,000원 |

### 예시 2: 퇴직금 1억원, 근속 20년, IRP 5,000만원

**입력:** 퇴직급여 = 100,000,000원, 근속 20년, IRP = 50,000,000원

| 단계 | 계산 | 결과 |
|------|------|------|
| 근속연수공제 | 15,000,000 + (20-10)×2,500,000 | 40,000,000원 |
| 환산급여 | floor((100,000,000 - 40,000,000) × 12 / 20) | 36,000,000원 |
| 환산급여별공제 | 8,000,000 + (36,000,000 - 8,000,000) × 60% | 24,800,000원 |
| 과세표준 | 36,000,000 - 24,800,000 | 11,200,000원 |
| 환산산출세액 | 11,200,000 × 6% | 672,000원 |
| 퇴직소득세 | floor(672,000 / 12 × 20) | 1,120,000원 |
| 지방소득세 | floor(1,120,000 × 10%) | 112,000원 |
| 이연퇴직소득세 | floor(1,120,000 × 50,000,000 / 100,000,000) | 560,000원 |
| 이연지방소득세 | floor(560,000 × 10%) | 56,000원 |
| 원천징수 소득세 | floor((1,120,000 - 560,000) / 10) × 10 | 560,000원 |
| 원천징수 지방소득세 | floor((112,000 - 56,000) / 10) × 10 | 56,000원 |
| 실수령액 | 100,000,000 - 560,000 - 56,000 | 99,384,000원 |
