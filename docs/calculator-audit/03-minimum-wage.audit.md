# 최저임금 검증 계산기 감사 시트

## 1. 개요

| 항목 | 내용 |
|------|------|
| 계산기명 | 최저임금 검증 (Minimum Wage Compliance) |
| 코드 파일 | `wage_calculator/calculators/minimum_wage.py` |
| 함수명 | `calc_minimum_wage(inp: WageInput, ow: OrdinaryWageResult) -> MinimumWageResult` |
| 적용 법조문 | 최저임금법 제5조 (최저임금액), 제6조 (최저임금의 효력), 제6조 제4항 (산입범위), 최저임금법 시행령 제5조의2 |
| 5인 미만 적용 여부 | 적용 (최저임금은 사업장 규모 무관, 1인 이상 전 사업장) |

수요: 741건 (28.5%) — 두 번째로 많은 임금 관련 질문 유형.

### 고정수당의 최저임금 산입 분류 키 (min_wage_type)

| 값 | 의미 | 산입 방법 |
|----|------|----------|
| `"standard"` (기본값) | 일반 고정수당 | 전액 산입 |
| `"regular_bonus"` | 정기상여금 | 제외율 적용 후 초과분만 산입 |
| `"welfare"` | 복리후생비 | 제외율 적용 후 초과분만 산입 |
| `"excluded"` | 연장/야간/휴일 수당 | 비산입 |

미설정 시 수당 이름 패턴으로 자동 추론 (minimum_wage.py:37-42):
- `_EXCLUDED_PATTERNS`: "연장", "야간", "휴일", "특근"
- `_BONUS_PATTERNS`: "상여금", "상여", "보너스", "격려금", "인센티브"
- `_WELFARE_PATTERNS`: "식대", "식비", "급식", "교통비", "교통", "차량", "통근", "주거", "숙박", "가족수당", "복리"

## 4. 계산 과정 (핵심)

### Step 1: 기준 연도 최저임금 조회

> minimum_wage.py:68-72

```
법정최저시급 = MINIMUM_HOURLY_WAGE[기준연도]
```
- 해당 연도 데이터 없으면 가장 최근 연도 사용 (경고 출력)

### Step 2: 수습기간 특례 적용

> minimum_wage.py:75-85

```
if 수습기간여부:
    법정최저시급 = 법정최저시급 x PROBATION_MIN_WAGE_RATE (0.9)
```
- 법적 근거: 최저임금법 제5조 제2항 — 근로계약 1년 이상이고 단순노무 아닌 경우, 수습 3개월 이내 90% 적용

### Step 3: 산입범위 제외율 조회

> minimum_wage.py:90-93

```
(상여금제외율, 복리후생제외율) = get_min_wage_inclusion_rates(기준연도)
법정최저월액기준 = 법정최저시급 x 209.0    # 제외기준 계산용 (시행령 제5조의2)
```
- 209시간 고정 사용 (법정 월 환산 기준)

### Step 4: 기본급 월액 산출

> minimum_wage.py:96, `_get_base_monthly()` (minimum_wage.py:229-243)

| 임금형태 | 환산 공식 | 코드 위치 |
|---------|----------|----------|
| `HOURLY` | 기본급월액 = 시급 x 월기준시간 | minimum_wage.py:232 |
| `DAILY` | 기본급월액 = (일급 / 1일소정근로시간) x 월기준시간 | minimum_wage.py:234-236 |
| `MONTHLY` | 기본급월액 = monthly_wage | minimum_wage.py:240 |
| `COMPREHENSIVE` | breakdown 있으면 base값, 없으면 monthly_wage | minimum_wage.py:238-239 |
| `ANNUAL` | 기본급월액 = 연봉 / 12 | minimum_wage.py:242 |

### Step 5: 고정수당 분류 및 산입액 계산

> minimum_wage.py:98-127

각 수당을 4가지 유형으로 분류한다:

```
for 수당 in 고정수당목록:
    월환산액 = _monthly_amount(수당)    # 분기/반기/연 → 월 환산
    산입유형 = _min_wage_type(수당)     # 명시 키 우선, 없으면 이름 패턴 추론

    if 산입유형 == "excluded":    → 비산입 (연장/야간/휴일)
    elif 산입유형 == "regular_bonus":  → 정기상여금 합산
    elif 산입유형 == "welfare":   → 복리후생비 합산
    else:                        → 전액산입 합산
```

**지급주기별 월환산** (`_monthly_amount()`, minimum_wage.py:246-256):

| payment_cycle / annual | 환산 | 코드 위치 |
|----------------------|------|----------|
| annual=True 또는 "연" | 금액 / 12 | minimum_wage.py:250-251 |
| "분기" | 금액 / 3 | minimum_wage.py:252-253 |
| "반기" | 금액 / 6 | minimum_wage.py:254-255 |
| "월" (기본) | 금액 그대로 | minimum_wage.py:256 |

### Step 6: 정기상여금/복리후생비 산입 계산

> minimum_wage.py:121-127

```
상여금제외기준액 = 법정최저월액기준 x 상여금제외율
상여금산입액 = max(0, 상여금합계 - 상여금제외기준액)

복리후생제외기준액 = 법정최저월액기준 x 복리후생제외율
복리후생산입액 = max(0, 복리후생합계 - 복리후생제외기준액)
```
- 2024년 이후: 제외율 = 0 → 전액 산입

### Step 7: 실질시급 산출 및 충족 판단

> minimum_wage.py:129-135

```
총산입월임금 = 기본급월액 + 전액산입수당 + 상여금산입액 + 복리후생산입액
실질시급 = 총산입월임금 / 월기준시간

충족여부 = (실질시급 >= 법정최저시급)
부족분시급 = max(0, 법정최저시급 - 실질시급)
부족분월액 = 부족분시급 x 월기준시간
```

## 7. 계산 예시

### 예시 1: 2025년 월급제 + 정기상여금 + 식대

**입력**:
- 기준연도: 2025
- 임금형태: 월급
- 월 기본급: 2,000,000원
- 월기준시간: 209h (통상임금 계산기에서 산출)
- 고정수당:
  - 정기상여금 300,000원/월 (min_wage_type: regular_bonus)
  - 식대 150,000원/월 (min_wage_type: welfare)
  - 야간수당 200,000원/월 (min_wage_type: excluded)

**계산 과정**:

Step 1 — 최저임금 조회:
```
법정최저시급 = MINIMUM_HOURLY_WAGE[2025] = 10,030원
```

Step 2 — 수습 아님 → 통과

Step 3 — 산입범위 제외율:
```
2025년 → get_min_wage_inclusion_rates(2025) = (0.0, 0.0)  # 전액 산입
법정최저월액기준 = 10,030 x 209 = 2,096,270원
```

Step 4 — 기본급 월액:
```
기본급월액 = 2,000,000원
```

Step 5 — 수당 분류:
```
정기상여금: 300,000원/월 → regular_bonus
식대: 150,000원/월 → welfare
야간수당: 200,000원/월 → excluded (비산입)
```

Step 6 — 산입 계산 (2025년: 전액 산입):
```
상여금제외기준 = 2,096,270 x 0.0 = 0원
상여금산입액 = max(0, 300,000 - 0) = 300,000원

복리후생제외기준 = 2,096,270 x 0.0 = 0원
복리후생산입액 = max(0, 150,000 - 0) = 150,000원
```

Step 7 — 실질시급:
```
총산입월임금 = 2,000,000 + 0 + 300,000 + 150,000 = 2,450,000원
실질시급 = 2,450,000 / 209 = 11,722원

충족: 11,722원 >= 10,030원 (초과 1,692원/h)
```

### 예시 2: 2023년 수습 + 정기상여금 제외율 적용

**입력**:
- 기준연도: 2023
- 임금형태: 월급
- 월 기본급: 1,900,000원
- 월기준시간: 209h
- 수습기간: True (3개월)
- 고정수당:
  - 분기상여금 600,000원/분기 (min_wage_type: regular_bonus)

**계산 과정**:

Step 1 — 최저임금 조회:
```
법정최저시급 = MINIMUM_HOURLY_WAGE[2023] = 9,620원
```

Step 2 — 수습 특례:
```
법정최저시급 = 9,620 x 0.9 = 8,658원
```

Step 3 — 산입범위 제외율:
```
2023년 → (0.05, 0.01)  # 상여금 5%, 복리후생 1%
법정최저월액기준 = 8,658 x 209 = 1,809,522원
```

Step 4 — 기본급:
```
기본급월액 = 1,900,000원
```

Step 5 — 수당 분류:
```
분기상여금: 600,000 / 3 = 200,000원/월 → regular_bonus
```

Step 6 — 상여금 산입:
```
상여금제외기준 = 1,809,522 x 0.05 = 90,476원
상여금산입액 = max(0, 200,000 - 90,476) = 109,524원
```

Step 7 — 실질시급:
```
총산입월임금 = 1,900,000 + 109,524 = 2,009,524원
실질시급 = 2,009,524 / 209 = 9,616원

충족: 9,616원 >= 8,658원 (초과 958원/h)
```
