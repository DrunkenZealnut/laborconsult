# 보상휴가 감사 시트

## 1. 개요

| 항목 | 내용 |
|------|------|
| 계산기명 | 보상휴가 환산 |
| 코드 파일 | `wage_calculator/calculators/compensatory_leave.py` |
| 함수명 | `calc_compensatory_leave(inp, ow)` |
| 적용 법조문 | 근로기준법 제57조 (보상휴가제) |
| 5인 미만 적용 여부 | 적용 (단, 가산수당 미적용으로 가산분 없음) |

## 2. 입력 항목

| 필드명(한글) | 필드명(영문) | 타입 | 기본값 | 필수여부 |
|-------------|-------------|------|--------|---------|
| 주 연장근로시간 | `inp.schedule.weekly_overtime_hours` | `float` | `0.0` | 선택 |
| 주 야간근로시간 | `inp.schedule.weekly_night_hours` | `float` | `0.0` | 선택 |
| 주 휴일근로시간 (8h 이내) | `inp.schedule.weekly_holiday_hours` | `float` | `0.0` | 선택 |
| 주 휴일근로 8h 초과분 | `inp.schedule.weekly_holiday_overtime_hours` | `float` | `0.0` | 선택 |

| 통상시급 | `ow.hourly_ordinary_wage` | `float` | - | `calc_ordinary_wage()` 결과 |

## 3. 적용 상수

| 상수명 | 값 | 단위 | 법적 근거 | 코드 위치 |
|--------|---|------|-----------|----------|

| `NIGHT_PREMIUM_RATE` | 0.5 | 배율 | 근기법 제56조 제3항 (야간근로 50% 가산) | `constants.py:37` |
| `HOLIDAY_RATE` | 0.5 | 배율 | 근기법 제56조 제2항 (휴일근로 8h이내 50% 가산) | `constants.py:38` |
| `HOLIDAY_OT_RATE` | 0.5 | 배율 | 근기법 제56조 제2항 (휴일근로 8h초과 추가 50%) | `constants.py:39` |
| `WEEKS_PER_MONTH` | 4.345 (365/7/12) | 주/월 | 실무 관행 (월 환산 계수) | `utils.py:8` |

## 4. 계산 과정 (핵심)

### Step 1: 사업장 규모별 배율 설정
```
if 5인미만:
    연장가산율 = 0.0,  야간가산율 = 0.0
    휴일가산율 = 0.0,  휴일초과가산율 = 0.0
else:
    연장가산율 = 0.5,  야간가산율 = 0.5
    휴일가산율 = 0.5,  휴일초과가산율 = 0.5
```
- 5인 미만 사업장: 근기법 제56조 적용 제외 -> 가산수당 없음
- 코드 참조: `compensatory_leave.py:51-63`
- 법적 근거: 근기법 제11조(적용 범위), 시행령 제7조(5인 미만 제외 조항)

### Step 2: 연장근로 보상휴가 시간 산출
```
연장근로보상휴가 = 주연장근로시간 x (1 + 연장가산율)
```
- 5인 이상: 1시간 연장 = 1.5시간 보상휴가 (실제 시간 + 가산분)
- 5인 미만: 1시간 연장 = 1.0시간 보상휴가 (가산 없음)
- 코드 참조: `compensatory_leave.py:72`
- 법적 근거: 근기법 제57조 + 제56조 제1항

### Step 3: 야간근로 보상휴가 시간 산출
```
야간근로보상휴가 = 주야간근로시간 x 야간가산율
```
- 야간근로는 가산분만 보상 (기본 시간은 소정근로에 포함)
- 5인 이상: 1시간 야간 = 0.5시간 보상휴가
- 5인 미만: 0시간 (가산 없음)
- 코드 참조: `compensatory_leave.py:74`
- 법적 근거: 근기법 제57조 + 제56조 제3항

### Step 4: 휴일근로 보상휴가 시간 산출 (8h 이내)
```
휴일근로보상휴가(8h이내) = 주휴일근로시간 x (1 + 휴일가산율)
```
- 5인 이상: 1시간 휴일 = 1.5시간 보상휴가
- 5인 미만: 1시간 휴일 = 1.0시간 보상휴가
- 코드 참조: `compensatory_leave.py:76`
- 법적 근거: 근기법 제57조 + 제56조 제2항 제1호

### Step 5: 휴일근로 보상휴가 시간 산출 (8h 초과)
```
휴일근로보상휴가(8h초과) = 주휴일초과시간 x (1 + 휴일가산율 + 휴일초과가산율)
```
- 5인 이상: 1시간 휴일초과 = 2.0시간 보상휴가
- 5인 미만: 1시간 휴일초과 = 1.0시간 보상휴가
- 코드 참조: `compensatory_leave.py:78`
- 법적 근거: 근기법 제57조 + 제56조 제2항 제2호

### Step 6: 총 보상휴가 시간 합산
```
총보상휴가 = 연장근로보상휴가 + 야간근로보상휴가 + 휴일근로보상휴가(8h이내) + 휴일근로보상휴가(8h초과)
```
- 코드 참조: `compensatory_leave.py:80`

### Step 7: 미사용 보상휴가 수당 산출 (주 기준)
```
미사용보상휴가수당(주) = 통상시급 x 총보상휴가
```
- 사용 기한 내 미사용 시 현금 지급 의무
- 코드 참조: `compensatory_leave.py:105`
- 법적 근거: 근기법 제57조 (미사용 시 금전 보상)

### Step 8: 월 환산
```
미사용보상휴가수당(월) = 미사용보상휴가수당(주) x WEEKS_PER_MONTH(4.345)
```
- 코드 참조: `compensatory_leave.py:111`

## 7. 계산 예시

### 예시 1: 5인 이상 사업장, 주 연장 10시간 + 야간 4시간 + 휴일 8시간

**입력:**
- 사업장 규모: `OVER_5`
- 통상시급: 15,000원
- 주 연장근로: 10시간
- 주 야간근로: 4시간
- 주 휴일근로(8h이내): 8시간
- 주 휴일초과: 0시간

**계산:**
1. 연장근로 보상휴가: 10 x (1 + 0.5) = 15.00시간
2. 야간근로 보상휴가: 4 x 0.5 = 2.00시간
3. 휴일근로 보상휴가: 8 x (1 + 0.5) = 12.00시간
4. 총 보상휴가: 15.00 + 2.00 + 12.00 = 29.00시간/주
5. 미사용 수당(주): 15,000 x 29.00 = 435,000원
6. 미사용 수당(월): 435,000 x 4.345 = 1,890,075원

**결과:**
- `compensatory_hours` = 29.00시간
- `unused_leave_pay` = 435,000원
- `monthly_unused_pay` = 1,890,075원

### 예시 2: 5인 미만 사업장, 주 연장 5시간 + 휴일 4시간

**입력:**
- 사업장 규모: `UNDER_5`
- 통상시급: 10,030원
- 주 연장근로: 5시간
- 주 야간근로: 0시간
- 주 휴일근로(8h이내): 4시간

**계산:**
1. 연장근로 보상휴가: 5 x (1 + 0.0) = 5.00시간 (가산 없음)
2. 야간근로 보상휴가: 0시간
3. 휴일근로 보상휴가: 4 x (1 + 0.0) = 4.00시간 (가산 없음)
4. 총 보상휴가: 5.00 + 0.00 + 4.00 = 9.00시간/주
5. 미사용 수당(주): 10,030 x 9.00 = 90,270원
6. 미사용 수당(월): 90,270 x 4.345 = 392,223원

**결과:**
- `compensatory_hours` = 9.00시간
- `unused_leave_pay` = 90,270원
- `monthly_unused_pay` = 392,223원
