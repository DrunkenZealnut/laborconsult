## 4. 계산 과정 (핵심)

### Step 1: 입력값 검증 및 보정
- 육아휴직 개월 수 ≤ 0 → 1개월로 보정 + 경고 (`parental_leave.py:92-94`)
- 12개월 초과 → 12개월로 제한 + 경고 (`parental_leave.py:96-98`)
- 공식: `휴직개월 = max(1, min(입력개월, 12))`
- 법적 근거: 고용보험법 제71조

### Step 2: 기본 육아휴직급여 산출 (80%)
- 공식: `기본급여 = 월통상임금 × 0.80`
- 상한/하한 적용: `월급여 = max(하한액, min(기본급여, 상한액))`
- 코드: `parental_leave.py:103-107`
- 법적 근거: 고용보험법 제70조

### Step 3: 사후지급금 계산 (25%)
- 공식: `사후지급금 = round(월급여 × 0.25)`
- 공식: `매월수령액 = round(월급여 - 사후지급금)` (월급여의 75%)
- 코드: `parental_leave.py:123-124`
- 법적 근거: 고용보험법 제70조 (복직 후 6개월 이상 근무 시 일괄 지급)

### Step 4: 아빠 보너스 계산 (두 번째 부모, 첫 3개월)
- 조건: `is_second_parent == True`
- 공식: `보너스급여 = min(월통상임금 × 1.00, 보너스상한액)`
- 보너스 적용 기간: `보너스개월 = min(3, 휴직개월)`
- 보너스 기간에도 사후지급금 25% 공제 적용
- 코드: `parental_leave.py:138-154`
- 법적 근거: 고용보험법 제70조 (같은 자녀에 대해 배우자 이후 두 번째 육아휴직)

### Step 5: 총 급여 합산
- 아빠 보너스 적용 시:
  - `보너스실수령 = round(보너스급여 × (1 - DEFERRED_RATIO))`
  - `총급여 = 보너스실수령 × 보너스개월 + 매월수령액 × (휴직개월 - 보너스개월)`
- 보너스 미적용 시:
  - `총급여 = 매월수령액 × 휴직개월`
- 코드: `parental_leave.py:157-165`

### Step 6: 총 사후지급금 합산
- 아빠 보너스 적용 시:
  - `보너스사후 = round(보너스급여 × DEFERRED_RATIO) × 보너스개월`
  - `일반사후 = 사후지급금 × (휴직개월 - 보너스개월)`
  - `총사후지급금 = 보너스사후 + 일반사후`
- 보너스 미적용 시:
  - `총사후지급금 = 사후지급금 × 휴직개월`
- 코드: `parental_leave.py:167-171`

### Step 7: 육아기 근로시간 단축급여 (선택)
- 조건: `reduced_work_hours_per_day > 0`
- 공식: `단축비율 = 단축시간 / 1일소정근로시간`
- 공식: `단축후월통상임금 = 월통상임금 × (1 - 단축비율)`
- 공식: `임금감소분 = 월통상임금 - 단축후월통상임금`
- 공식: `단축급여 = 임금감소분 × 0.80`
- 코드: `parental_leave.py:182-199`
- 법적 근거: 고용보험법 제73조의2

## 6. 예외 처리 / 엣지 케이스

| 조건 | 처리 | 법적 근거 |
|------|------|-----------|
| `parental_leave_months <= 0` | 1개월로 보정 + 경고 | `parental_leave.py:92-94` |
| `parental_leave_months > 12` | 12개월로 제한 + 경고 | 고용보험법 제71조 (최대 12개월) |
| 통상임금 × 80% > 상한액 | 상한액(150만원) 적용 | 고용노동부 고시 |
| 통상임금 × 80% < 하한액 | 하한액(70만원) 적용 | 고용노동부 고시 |
| `is_second_parent=True`, 보너스 100% > 보너스 상한 | 보너스 상한(250만원) 적용 | 고용노동부 고시 |
| 기준 연도가 `PARENTAL_LEAVE_LIMITS`에 없음 | 2025년 기준 fallback | `parental_leave.py:83` |
| `daily_work_hours == 0` (단축급여 계산 시) | if문으로 division by zero 방지 | `parental_leave.py:190` |
| 아빠 보너스 + 사후지급금 병행 | 보너스 기간에도 동일하게 25% 공제 적용 | `parental_leave.py:161` |
| `getattr` 방어 | 필드 미존재 시 기본값 반환 | `parental_leave.py:88-90` |

## 7. 계산 예시

### 예시 1: 기본 육아휴직 (월 통상임금 300만원, 6개월)

**입력:** 월통상임금 = 3,000,000원, 휴직 6개월, 아빠보너스 미적용

| 단계 | 계산 | 결과 |
|------|------|------|
| 기본급여 | 3,000,000 × 80% | 2,400,000원 |
| 상한 적용 | min(2,400,000, 1,500,000) | 1,500,000원 (상한 적용) |
| 사후지급금 | round(1,500,000 × 25%) | 375,000원/월 |
| 매월 수령액 | 1,500,000 - 375,000 | 1,125,000원/월 |
| 총 수령액 | 1,125,000 × 6 | 6,750,000원 |
| 총 사후지급금 | 375,000 × 6 | 2,250,000원 |

### 예시 2: 아빠 보너스 적용 (월 통상임금 200만원, 12개월)

**입력:** 월통상임금 = 2,000,000원, 12개월, `is_second_parent=True`

| 단계 | 계산 | 결과 |
|------|------|------|
| 기본급여(80%) | 2,000,000 × 80% = 1,600,000 | → 상한 1,500,000원 |
| 보너스급여(100%) | min(2,000,000, 2,500,000) | 2,000,000원 |
| 보너스 실수령(75%) | round(2,000,000 × 75%) | 1,500,000원/월 × 3개월 |
| 일반 실수령(75%) | round(1,500,000 × 75%) | 1,125,000원/월 × 9개월 |
| 총 수령액 | 1,500,000×3 + 1,125,000×9 | 14,625,000원 |
| 보너스 사후지급금 | round(2,000,000 × 25%) × 3 | 1,500,000원 |
| 일반 사후지급금 | 375,000 × 9 | 3,375,000원 |
| 총 사후지급금 | 1,500,000 + 3,375,000 | 4,875,000원 |
