# 퇴직금 계산기 감사 시트

## 1. 개요

| 항목 | 내용 |
|------|------|
| 계산기명 | 퇴직금 계산기 |
| 코드 파일 | `wage_calculator/calculators/severance.py` |
| 함수명 | `calc_severance(inp: WageInput, ow: OrdinaryWageResult) -> SeveranceResult` |
| 적용 법조문 | 근로자퇴직급여보장법 제4조·제8조·제9조, 근로기준법 제2조 (평균임금), 근기법 시행령 제2조, 대법원 2023다302579 |
| 5인 미만 적용 여부 | 적용 (근퇴법은 사업장 규모 불문). 코드에서 안내 경고 출력 (L199~204) |

**주요 내부 함수:**
- `_calc_avg_daily_1y()` (L223~237) -- 퇴직 전 1년 평균임금 산정
- `_ineligible()` (L240~263) -- 미발생 결과 반환 헬퍼
- `calc_average_wage()` (외부 모듈: `average_wage.py`) -- 3개월 평균임금·통상임금 환산 일급 산정

## 4. 계산 과정 (핵심)

### Step 1: 재직기간 산정 (L63~74)

```
입사일 = parse_date(inp.start_date)
퇴직일 = parse_date(inp.end_date) or date.today()
계속근로일수 = (퇴직일 - 입사일).days
계속근로연수 = 계속근로일수 / 365
```

- **법적 근거**: 근퇴법 제4조
- **코드 참조**: `severance.py` L63~74

### Step 2: 자격 요건 확인 (L76~106)

#### 2-A: 계속근로 1년 이상 확인

```
if 계속근로일수 < 365:
    → 퇴직금 미발생 (SeveranceResult.is_eligible = False)
```

- **법적 근거**: 근퇴법 제4조 제1항
- **코드 참조**: `severance.py` L78~85

#### 2-B: 일용직 특례 판단

```
if 근무형태 == DAILY_WORKER:
    if 월평균근무일수 < 4:
        → 경고: 계속 근무 인정 어려움
    else:
        → 경고: 퇴직금 수급 가능성 있음
```

- **법적 근거**: 대법원 2024.3.12. 선고 2023다302579
- **코드 참조**: `severance.py` L88~106

### Step 3: 평균임금 산정 - 3가지 비교 (L108~151)

#### 3-A: 3개월 평균임금 (기본)

`calc_average_wage()` 모듈 호출 (`average_wage.py` L38~135):

```
3개월임금총액 = sum(최근3개월임금) (또는 월급 × 3 추정)
상여금가산 = 연간상여금총액 × 3 / 12
연차수당가산 = 미사용연차수당 × 3 / 12
총액 = 3개월임금총액 + 상여금가산 + 연차수당가산

3개월평균임금 = 총액 / 산정기간일수
```

- **법적 근거**: 근기법 제2조 제1항 (평균임금 정의)
- **코드 참조**: `severance.py` L111~112, `average_wage.py` L38~135

#### 3-B: 1년 평균임금 (대법원 2023다302579)

```
if 최근1년임금 제공:
    1년평균임금 = sum(최근1년임금) / 1년일수
elif 월급 제공:
    1년평균임금 = 월급 × 12 / 365
else:
    1년평균임금 = 월통상임금 × 12 / 365
```

- **법적 근거**: 대법원 2024.3.12. 선고 2023다302579 (1년 평균임금 유리 원칙)
- **코드 참조**: `_calc_avg_daily_1y()` L223~237

#### 3-C: 통상임금 환산 일급

```
통상임금환산일급 = 통상시급 × 8
```

- **법적 근거**: 근기법 시행령 제2조 (평균임금 < 통상임금 → 통상임금 적용)
- **코드 참조**: `average_wage.py`에서 `ow.daily_ordinary_wage` 반환

### Step 4: 유리한 기준 선택 (L120~145)

```
후보 = {"3개월": 3개월평균임금, "통상임금": 통상임금환산일급}

# 1년 평균임금이 3개월보다 5% 초과 높으면 후보에 추가
if 1년평균임금 > 0 and 1년평균임금 > 3개월평균임금 × 1.05:
    후보["1년"] = 1년평균임금

# 통상임금이 3개월 평균보다 높으면 경고
if 통상임금환산일급 > 3개월평균임금:
    → 통상임금 기준 적용 안내

적용기준 = max(후보, key=금액)
적용평균임금 = 후보[적용기준]
```

- **법적 근거**: 대법원 2023다302579, 근기법 시행령 제2조
- **코드 참조**: `severance.py` L120~145

### Step 5: 퇴직금 산정 (L153~158)

```
퇴직금 = 적용평균임금 × 30일 × (계속근로일수 / 365)
```

- **법적 근거**: 근퇴법 제8조 제1항
- **코드 참조**: `severance.py` L154

## 7. 계산 예시

### 예시 1: 일반 정규직 3년 근무 (월급 300만원, 상여금 연 600만원)

**입력:**
- 입사일: 2023-03-01
- 퇴직일: 2026-03-01
- 월급: 3,000,000원
- 통상시급: 14,354원 (월 통상임금 3,000,000원 / 209시간)
- 연간 상여금: 6,000,000원
- 미사용 연차수당: 0원

**계산 과정:**

```
계속근로일수 = (2026-03-01 - 2023-03-01).days = 1096일
계속근로연수 = 1096 / 365 = 3.00년 (≥ 1년 → 자격 충족)

[3개월 평균임금]
3개월임금총액 = 3,000,000 × 3 = 9,000,000원
상여금가산 = 6,000,000 × 3/12 = 1,500,000원
총액 = 9,000,000 + 1,500,000 = 10,500,000원
산정기간일수 = 92일 (기본값)
3개월평균임금 = 10,500,000 / 92 = 114,130원/일

[1년 평균임금]
1년평균임금 = 3,000,000 × 12 / 365 = 98,630원/일

[통상임금 환산일급]
통상임금환산일급 = 14,354 × 8 = 114,832원/일

[유리한 기준 선택]
3개월(114,130) vs 통상임금(114,832) → 통상임금 적용
1년(98,630)은 3개월(114,130) × 1.05 = 119,837보다 낮으므로 후보 미추가

적용평균임금 = 114,832원/일 (통상임금 기준)

[퇴직금]
퇴직금 = 114,832 × 30 × (1096 / 365) = 10,334,880원
```

**결과:** 퇴직금 10,334,880원, 적용 기준: 통상임금

### 예시 2: 일용직 2년 근무 (변동급여, 3개월 < 1년 평균)

**입력:**
- 입사일: 2024-03-01
- 퇴직일: 2026-03-01
- 근무형태: 일용직
- 월 평균 근무일수: 10일
- 최근 3개월 임금: [1,800,000, 1,500,000, 1,200,000]
- 최근 1년 임금: [2,500,000, 2,800,000, 2,600,000, 2,400,000, 2,300,000, 2,500,000, 2,700,000, 2,200,000, 2,000,000, 1,800,000, 1,500,000, 1,200,000]
- 통상시급: 10,030원

**계산 과정:**

```
계속근로일수 = 731일 (≥ 365 → 자격 충족)

[일용직 판단]
월 평균 10일 → "퇴직금 수급 가능성 있음" (대법원 2023다302579)

[3개월 평균임금]
3개월임금총액 = 1,800,000 + 1,500,000 + 1,200,000 = 4,500,000원
3개월평균임금 = 4,500,000 / 92 = 48,913원/일

[1년 평균임금]
1년임금총액 = 26,500,000원
1년평균임금 = 26,500,000 / 365 = 72,603원/일

[비교]
72,603 > 48,913 × 1.05 (= 51,359) → 1년 기준 적용

[통상임금 환산일급]
통상임금환산일급 = 10,030 × 8 = 80,240원/일

[유리한 기준]
3개월(48,913) vs 1년(72,603) vs 통상임금(80,240)
→ 통상임금 80,240원/일 적용

[퇴직금]
퇴직금 = 80,240 × 30 × (731 / 365) = 4,814,400원
```

**결과:** 퇴직금 4,814,400원, 적용 기준: 통상임금
